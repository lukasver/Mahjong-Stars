# マーケットプレイス

ゲーム内マーケットプレイスは、プレイヤーがキャラクター、チャーム、スタジオ、AIモジュールを売買する場所です。機能には、ピアツーピア取引、AI-as-a-Service、さらには物理的なコレクティブルの交換が含まれます。

<Callout type="info">
**オプション機能**: マーケットプレイスにはブロックチェーン統合が必要です。従来のゲームプレイを好むプレイヤーは、これを完全に無視できます。すべての資産は標準的なゲーム内アイテムとして完全に機能します。
</Callout>

---

## 取引と流動性

マーケットプレイスは、資産取引のための安全で透明性の高いプラットフォームを提供します。

### リスティングツール

売り手は、資産をリスティングするための柔軟なオプションを備えています。

**固定価格リスティング**:
- トークンで価格を設定
- 購入者が承諾すれば即時販売
- 期限なし（キャンセルまたは販売されるまで）
- 価格が確立された一般的なアイテムに最適
- リスクが最も低く、予測可能な結果

**オークションリスティング**:
- **イングリッシュオークション（上昇）**:
  - 開始入札額と最低落札価格を設定
  - 購入者が互いに入札
  - オークション終了時に最高入札額が勝利
  - 通常24〜72時間
  - 希少で高価なアイテムに最適

- **ダッチオークション（下降）**:
  - 高価格から開始し、時間とともに減少
  - 現在の価格を承諾した最初の購入者が勝利
  - 定期的に価格が下落（例：毎時間）
  - 購入者に緊急性を生み出す
  - 在庫を迅速に移動させるのに適している

**バンドルリスティング**:
- 複数のアイテムをまとめて販売
- パッケージ取引を提供する
- 購入者にとってより良い価値
- 関連アイテムの迅速な販売
- カスタムバンドル価格

**リスティング手数料**:
- リスティングのための少額の手数料（価値の0.5〜1％、最大100トークン）
- アイテムが売れなかった場合に返金（オプション）
- スパムリスティングを防ぐ
- ステーキング者への手数料削減

<Callout type="success">
**低い障壁**: 最小限のリスティング手数料により、誰でもマーケットプレイス経済に参加できます。
</Callout>

### 検索とフィルター

購入者は、探しているものを簡単に見つけることができます。

**検索機能**:
- **テキスト検索**: 名前、説明、特性で検索
- **高度なフィルター**:
  - アセットタイプ（キャラクター、チャーム、スタジオ、AIモジュール）
  - 希少性ティア（コモンからレジェンダリー）
  - レベル範囲
  - 価格範囲
  - 販売者評価
  - リスティングタイプ（オークション対固定）
  - リスティング日（新しい順など）
- **並べ替えオプション**:
  - 価格（安い順、高い順）
  - 人気（最も表示された）
  - 終了間近（オークション）
  - 最近リスティングされた
  - ベストディール（価値評価）

**スマートフィルター**:
- コレクションに基づいたAI推奨アイテム
- 類似アイテム（リスティング表示時）
- 価格履歴とトレンド
- 比較分析ツール
- ウォッチリストアイテムアラート

**トレンドセクション**:
- 今週最も取引されたアイテム
- 値上がりのある資産
- 人気のある検索
- コミュニティのお気に入り
- シーズナルハイライト

### 安全なエスクロー

すべての取引はスマートコントラクトを通じて処理され、安全性を確保します。

**取引のセキュリティ**:
1. 購入者がエスクローコントラクトに支払いを行う
2. 販売者がエスクローに資産を転送する
3. スマートコントラクトが両側を確認する
4. 同時交換が実行される
5. 両当事者が原子的にアイテムを受け取る

**保護機能**:
- **詐欺なし**: アイテムを与えずに金銭を奪うことはできません
- **チャージバックなし**: ブロックチェーン取引は最終的です
- **紛争解決**: 問題に対するプラットフォーム仲裁
- **評価システム**: 販売者/購入者の評判を築く
- **購入者保護**: 最終決定前に資産を確認する

**スマートコントラクトの利点**:
- トラストレスな取引
- 透明性の高いコード
- 監査済みのセキュリティ
- 最小限の手数料（ガスのみ）
- 即時決済

### 分析ダッシュボード

データに基づいた取引の意思決定:

**過去の価格**:
- すべてのアイテムの価格チャート
- 取引量追跡
- 平均販売価格
- フロア価格（最低利用可能）
- シーリング価格（最高販売価格）

**市場トレンド**:
- 供給と需要の指標
- 価格モメンタム
- 取引速度
- ホルダー分布
- 希少性乗数

**ポートフォリオ追跡**:
- コレクションの現在の価値
- 購入価格対現在の価値
- 損益計算
- パフォーマンスの高いアイテム
- 投資分析

**アラート**:
- 値下がり通知
- 欲しいアイテムのリスティングアラート
- 入札超過通知
- オークション終了リマインダー
- ポートフォリオ価値の変更

<Callout type="warning">
**市場インテリジェンス**: 分析を使用して、投機的なギャンブルではなく、情報に基づいた取引の決定を行ってください。使用して楽しむアイテムに焦点を当ててください。
</Callout>

## 化粧品アイテムと仮想資産

ゲームプレイ資産を超えて、マーケットプレイスは純粋な化粧品アイテムを提供します。

### 化粧品カテゴリ

**ビジュアルカスタマイズ**:
- **テーブルスキン**: カスタムテーブルデザインと素材
- **タイルセット**: ユニークなタイル表示
- **背景**: ゲーム環境テーマ
- **アバターフレーム**: プロフィール画像ボーダー
- **エモートとステッカー**: 表現パック
- **勝利アニメーション**: お祝いエフェクト

**オーディオカスタマイズ**:
- **ミュージックパック**: バックグラウンドサウンドトラック
- **効果音**: カスタムタイルサウンド
- **ボイスパック**: キャラクターダイアログオプション
- **環境音**: 環境オーディオ

**限定版**:
- **シーズナルアイテム**: ホリデーおよびイベント限定
- **コラボアイテム**: ブランドパートナーシップ
- **アーティストシリーズ**: アーティストによる限定版
- **記念アイテム**: 特別イベント記念品
- **創設者アイテム**: 初期サポーター限定

### 化粧品販売モデル

**一次販売**:
- 新しい化粧品が定期的にリリースされる
- 希少アイテムの数量限定
- プラットフォームからの固定価格
- 売上の一部は開発をサポート

**二次市場**:
- プレイヤーが化粧品を再販する
- 供給と需要によって決定される価格
- プラットフォームが少額の手数料を取る
- 元のクリエイターがロイヤリティを受け取る場合がある（オプション）

**シーズナルリリース**:
- 四半期ごとのテーマ別コレクション
- 期間限定
- シーズン後の希少性が価値を駆動する
- コレクターの魅力

## AI-as-a-Serviceと収益化の可能性

Mahjong Stars独自の革新的な機能：AIマーケットプレイス。

### AIレンタル

十分にトレーニングされたAIコンパニオンを持つプレイヤーは、それを他者に貸し出すことができます。

**レンタルメカニズム**:
- **時間単位レンタル**: 10〜100トークン/時間（オーナー設定）
- **日単位レンタル**: 100〜1,000トークン/日
- **試合ベース**: 50〜500トークン/試合
- **サブスクリプション**: 週次/月次レンタルパッケージ

**レンタル機能**:
- レンタル者は試合でAIを使用する
- 収益はレンタル者に行く（AIオーナーはレンタル料を得る）
- AIのパフォーマンスが追跡される
- AI品質の評価システム
- キャンセルポリシー

**オーナーのメリット**:
- トレーニング投資からの受動的収入
- プラットフォームが5〜10％の手数料を取る
- AIトレーナーとしての評判を築く
- 需要が高いほど価格が高くなる
- 複数の同時レンタル（許可されている場合）

**パフォーマンス保証**:
- 広告された勝率は正確である必要がある
- パフォーマンスの低いAIに対する返金
- 検証システム
- コミュニティレビュー
- 紛争解決

<Callout type="success">
**革新的な経済**: よくトレーニングされたAIは、トレーナーに継続的な収入を生み出す貴重な資産になります。
</Callout>

### AIコーチングサービス

経験豊富なプレイヤーは、AIモデルを使用して1対1のコーチングを提供できます。

**コーチングオプション**:
- **戦略セッション**: AIの洞察を用いたリプレイ分析（500〜5,000トークン/時間）
- **トレーニングプラン**: カスタムAIトレーニングロードマップ（1,000〜10,000トークン）
- **ライブコーチング**: ゲーム中の観戦とアドバイス（1,000〜10,000トークン/セッション）
- **AI最適化**: パフォーマンスのためのAI設定の調整（500〜5,000トークン）

**予約システム**:
- マーケットプレイスを通じてスケジュール設定
- エスクローが両当事者を保護
- 評価とレビューシステム
- キャンセルポリシー
- セッション録画（許可を得て）

**コーチのメリット**:
- 専門知識を収益化する
- 個人のブランドを構築する
- コーチングビジネスを成長させる
- プラットフォームプロモーション（トップコーチが特集される）
- コミュニティでの認知

### AIテンプレートとモジュール

開発者は、特定の戦略に特化したAIテンプレートを作成できます。

**AI製品**:
- **バリアントスペシャリスト**: 特定のルールセットに最適化されたAI（1,000〜10,000トークン）
- **戦略モジュール**: 防御的、攻撃的、バランスの取れたスタイル（500〜5,000トークン）
- **トレーニングアクセラレーター**: より速い学習のためのデータセット（250〜2,500トークン）
- **分析ツール**: 高度な試合後分析（500〜5,000トークン）
- **ハイブリッドモジュール**: 人間とAIの戦略の組み合わせ（1,000〜10,000トークン）

**開発者の経済**:
- 独自の価格設定
- プラットフォーム手数料（10〜15％）
- 自動更新（オプション）
- カスタマーサポートの責任
- 収益追跡ダッシュボード

**品質管理**:
- 承認前のプラットフォームテスト
- ユーザー評価とレビュー
- パフォーマンスベンチマーク
- 返金ポリシー
- 特集開発者プログラム

## 物理コレクティブル交換

Mahjong Starsならでは：デジタル経済にリンクされた物理アイテムを取引します。

### 物理コレクティブル

時折リリースされる限定版の物理グッズ：

**アイテムの種類**:
- **テーマ別マグカップ**: キャラクターデザイン、限定版
- **アートプリント**: プラットフォームのアートワーク、サイン入りエディション
- **アパレル**: 排他的なデザインのTシャツ、パーカー
- **麻雀セット**: 特別なデザインの物理的なタイルセット
- **記念アイテム**: イベント記念品

**入手方法**:
- ゲーム内ショップからソフト通貨で購入
- 数量限定
- 先着順
- 最低アカウントレベルが必要な場合がある
- 確認済みの住所に発送

<Callout type="info">
**デジタルとフィジカルの架け橋**: 物理的なコレクティブルは、デジタルプラットフォームとの具体的なつながりを作り出し、コミュニティエンゲージメントを高めます。
</Callout>

### 交換メカニズム

プレイヤーは、取得した物理アイテムを再出品できます。

**リスティングプロセス**:
1. 物理コレクティブルを所有している（あなたに発送される）
2. 専用の物理交換に出品する
3. 希望小売価格を設定する（ソフト通貨またはトークン）
4. 他のユーザーが入札する
5. オークションが終了し、最高入札者が勝利する
6. 勝者が支払い、あなたがアイテムを発送する
7. 配達確認で支払いがリリースされる

**オークション形式**:
- 最低入札額要件
- オークション期間（通常7〜14日）
- 最低落札価格はオプション
- 即時購入オプション
- 送料の透明性

**安全機能**:
- 確認済みの住所
- 追跡必須
- 配達確認までエスクロー
- 紛争解決
- 販売者評価が重要

**プラットフォームの役割**:
- リスティングをホストする（物理アイテムの手数料はなし）
- 紛争を仲裁する
- 正規品を検証する
- ガイドラインを提供する
- コミュニティによる管理

## マーケットプレイス手数料

プラットフォームを維持し、価値を提供するため：

**取引手数料**:
- **デジタル資産**: 10％の基本手数料
- **ステーカ​​ー割引**: 10〜60％の手数料削減（ステーキングティアを参照）
- **AIレンタル**: 5〜10％の手数料
- **AI製品**: 10〜15％の手数料
- **物理アイテム**: 0％の手数料（コミュニティ主導）

**手数料の分配**:
- 40％をバーン（デフレ）
- 30％をステーカ​​ーへ（報酬）
- 20％を開発ファンドへ
- 10％を流動性プロバイダーへ

**ガス料金**:
- ブロックチェーン取引コスト（ユーザー負担）
- 通常は少量（数セント〜数ドル）
- ネットワークの混雑状況に基づく動的
- 新規ユーザーへの補助金プログラム（最初の取引は無料）
