# プレイヤーの進行と維持

進行は、長期的なエンゲージメントの不可欠な要素です。「Mahjong Stars」では、プレイヤーのモチベーションとエンゲージメントを維持するために、複数のレベルアップシステム、報酬メカニズム、競争構造が組み込まれています。

---

## レベルアップシステムとAIの成長

「Mahjong Stars」には、継続的な進歩の機会を提供する相互接続された進行システムが備わっています。

### プレイヤーレベル

全体的な経験を反映し、プレイヤーが進むにつれて新しい機能がアンロックされます。

**レベル進行**:
- **開始レベル**: 1（すべてのプレイヤーがここから開始します）
- **最大レベル**: 100（拡張の余地あり）
- **経験値（XP）の獲得源**: プレイした対戦、完了したクエスト、獲得した実績
- **レベル要件**: 指数関数的に増加するXPが必要

**経験値（XP）の獲得**:
- **対戦完了**: 50～200 XP（パフォーマンスに基づく）
- **勝利**: ボーナス 100～300 XP
- **パーフェクトゲーム**: 追加 500 XP
- **デイリークエスト**: 各 200～1,000 XP
- **ウィークリーチャレンジ**: 1,000～5,000 XP
- **実績**: 500～10,000 XP
- **トーナメント参加**: 1,000～50,000 XP

**レベル別の機能アンロック**:

| レベル範囲 | アンロックされる機能 |
|------------|-------------------|
| 1～10 | 基本的なゲームプレイ、チュートリアル、カジュアル対戦 |
| 11～20 | ランキングプレイ、最初のAIスロット、デイリークエスト |
| 21～30 | 高度なチュートリアル、トーナメントアクセス、クランへの参加 |
| 31～40 | スタジオアクセス（所有している場合）、AIレンタル機能、カスタムルーム |
| 41～50 | 高度なトーナメント、クラン作成、マーケットプレイスでの販売 |
| 51～60 | ハイステークステーブル、プレミアム機能、メンターシッププログラム |
| 61～70 | 特別トーナメント、早期機能アクセス |
| 71～80 | プレステージオプション、レジェンダリーアイテムアクセス |
| 81～90 | エリート競技、プラットフォームテストプログラム |
| 91～100 | 最大限の特典、特別な認識、限定コンテンツ |

**レベルアップ報酬**:
- ソフト通貨ボーナス
- チャーム報酬
- キャラクターの欠片
- ラッフルチケット
- コスチュームアンロック
- タイトルオプション

<Callout type="success">
**常に進行中**: 最大レベルのプレイヤーでも、プレステージシステムやシーズンリセットを通じて報酬を獲得し続けます。
</Callout>

### キャラクターレベル

各キャラクターは対戦を通じて経験値を蓄積し、追加の機能と特典をアンロックします。

**キャラクター進行**:
- **個別のXP**: 各キャラクターは個別にレベルアップします
- **アクティブキャラクターがXPを獲得**: 装備中のキャラクターのみ
- **最大レベル**: キャラクターごとに50
- **プレステージオプション**: 永続的なボーナスを得てレベル1にリセット

**キャラクターXP獲得源**:
- 対戦でのキャラクター使用（主）
- キャラクターでの勝利（ボーナス）
- 特定のキャラクターチャレンジ
- キャラクター固有の実績
- イベント参加

**レベルアップ特典**:
- **コスチュームオプション**: 特定のレベルでアンロック
- **エモート**: キャラクター固有の表現（レベル5、15、25、35、45）
- **AI効率**: AIトレーニング速度の向上（レベルごとに+2%）
- **ステータスボーナス**: マイナーな属性増加
- **スキルポイント**: スキルツリーに割り当て（レベルごとに1ポイント）

**スキルツリーの割り当て**:

**戦略ブランチ**:
- AIトレーニング速度ボーナス
- 追加AIスロットのアンロック（レベル20、40）
- ゲーム後分析の強化
- 戦略ヒントの改善
- 確率計算支援

**ソーシャルブランチ**:
- ユニークなエモートとステッカー
- チャットカスタマイズオプション
- フレンドボーナス乗数
- クラン貢献度の増加
- ソーシャルXPボーナス

**経済ブランチ**:
- ソフト通貨獲得ボーナス（ポイントごとに+1%）
- ラッフルチケット乗数
- トーナメント参加費割引（最大25%）
- マーケットプレイス手数料削減
- チャームドロップ率の増加

**キャラクタープレステージ**:
- レベル50で利用可能
- レベル1にリセットしますが、スキルポイントは保持
- 永続的なステータスボーナスを獲得
- 視覚的なプレステージインジケーター
- 限定コスチュームオプション
- 複数のプレステージレベルが可能

### AIレベル

AIコンパニオンは、より多くの対戦をプレイするにつれてレベルアップし、パフォーマンスが向上します。

**AI進行メカニズム**:
- **データベースのレベルアップ**: より多くのゲームプレイデータで向上
- **対戦経験**: すべての対戦がAIの成長に貢献
- **最大レベル**: 50（AIインスタンスあたり）
- **品質ティア**: ブロンズ、シルバー、ゴールド、プラチナ、ダイヤモンド

**AIレベル特典**:
- **より速い決定**: シンクタイムの短縮（5～50%高速化）
- **より良いリスク評価**: 確率計算の向上
- **適応力の強化**: 対戦相手への迅速な適応
- **戦略の洗練**: より高度なプレイパターン
- **一貫性**: より信頼性の高いパフォーマンス

**AIトレーニング要件**:
- **初期トレーニング**: 基本的な機能には50～100ゲーム
- **習熟**: 信頼性の高いパフォーマンスには200～500ゲーム
- **優秀**: トップティアAIには1,000ゲーム以上
- **メンテナンス**: 継続的なプレイでAIを最新の状態に保つ
- **再トレーニング**: 新しい戦略でAIを更新

**AIトレーニングの加速**:
- **スタジオトレーニング施設**: より高速なトレーニング
- **トレーニングモジュール**: 購入可能なXPブースト
- **シミュレーション対戦**: AI経験値としてカウント
- **ターゲットトレーニング**: 特定の弱点に焦点を当てる
- **一括トレーニング**: 複数のAIを同時にトレーニング（スタジオ機能）

<Callout type="info">
**生きているAI**: あなたのAIは、プレイするにつれて継続的に進化し、メタの変化に適応し、蓄積された経験に基づいて戦略を洗練します。
</Callout>

## 報酬とロイヤルティプログラム

頻繁なプレイを奨励するために、「Mahjong Stars」は包括的な報酬システムを提供しています。

### デイリークエスト

通常のゲームプレイで完了する簡単な目標。

**クエストの種類**:
- **X回対戦に勝利**: 3回対戦に勝利（報酬: 1,000ソフト通貨）
- **Yゲームプレイ**: 結果に関係なく5ゲームを完了（報酬: 500ソフト通貨）
- **特定キャラクターの使用**: 3ゲームをフィーチャーキャラクターで使用（報酬: キャラクターXPブースト）
- **ソーシャルクエスト**: 友達とプレイ、ギフトを送信（報酬: ソーシャルアイテム）
- **バリアントクエスト**: 特定の麻雀バリアントをプレイ（報酬: バリアント固有の報酬）

**クエストシステム**:
- **デイリーリセット**: 24時間ごとに新しいクエスト
- **クエストスロット**: 同時に3～5個のアクティブなクエスト
- **リロールオプション**: 1日1回クエストを交換（ソフト通貨で可能）
- **連続ボーナス**: 連続した日数で報酬が倍増
- **クイック完了**: ほとんどのクエストは30～60分で達成可能

### ウィークリーチャレンジ

戦略的な計画を必要とする、より長期的な目標。

**チャレンジ例**:
- **特定の手役で勝利**: 特定の役で勝利（3回）
- **ランクの目標達成**: ランキングラダーを登る（報酬: ランクベースの賞品）
- **トーナメントパフォーマンス**: ウィークリートーナメントでトップ50に入る
- **コレクション目標**: 5つの新しいチャームを獲得
- **AIトレーニング**: AIで50ゲームプレイ（報酬: AI XPブースト）

**ウィークリーチャレンジ特典**:
- **より大きな報酬**: チャームバンドル、キャラクターの欠片、かなりのソフト通貨
- **戦略的な深み**: 計画と献身が必要
- **進行フォーカス**: 特定のプレイスタイルを奨励
- **コミュニティイベント**: グローバル目標と連動する場合あり

### ロイヤルティティア

プレイヤーは、連続プレイ日数と総対戦数に基づいてロイヤルティポイントを蓄積します。

**ロイヤルティティアシステム**:

| ティア | 要件 | 特典 |
|---|---|---|
| **ブロンズ** | 7連続日数、50総対戦 | +5%ソフト通貨、週1ラッフルチケット |
| **シルバー** | 30連続日数、200総対戦 | +10%ソフト通貨、週2ラッフルチケット、トーナメント割引（10%） |
| **ゴールド** | 90連続日数、500総対戦 | +15%ソフト通貨、週3ラッフルチケット、トーナメント割引（15%）、限定チャーム |
| **プラチナ** | 180連続日数、1,000総対戦 | +20%ソフト通貨、週5ラッフルチケット、トーナメント割引（20%）、優先サポート |
| **ダイヤモンド** | 365連続日数、2,500総対戦 | +25%ソフト通貨、週10ラッフルチケット、トーナメント割引（25%）、限定アクセス |

**ロイヤルティ特典**:
- **ソフト通貨乗数**: 永続的な獲得増加
- **手数料削減**: トーナメント参加費割引
- **ラッフルアクセス**: 限定ラッフルへの自動エントリー
- **優先機能**: 新コンテンツへの早期アクセス
- **コスチューム報酬**: ロイヤルティ限定アイテム
- **ステータス認識**: 特別なバッジとタイトル

**ロイヤルティの維持**:
- 毎日のログインで十分（試合は不要）
- 欠席日に対する猶予期間（上位ティアでは2日間のバッファ）
- バケーションモード（ロイヤルティを一時停止、ゴールド以上で利用可能）
- ストリークが失われた場合のカムバックボーナス

<Callout type="warning">
**ストリーク保護**: 上位のロイヤルティティアは欠席日に対する保護を提供しますが、最大の特典を得るには一貫性の維持が鍵となります。
</Callout>

### イベントパス

シーズンイベントでは、進行報酬付きのイベントパスが導入されます。

**イベントパス構造**:
- **無料ティア**: すべてのプレイヤーが利用可能、控えめな報酬
- **プレミアムティア**: オプション購入、強化された報酬
- **期間**: 通常、シーズンごとに4～8週間
- **レベル**: パスごとに50～100レベル

**イベントXPの獲得**:
- 対戦のプレイ（どのモードでも）
- イベント固有のクエストの完了
- テーマ別トーナメントへの参加
- 毎日のログインボーナス
- 特別イベントチャレンジ

**無料ティア報酬**（例）:
- ソフト通貨バンドル
- コモンチャーム
- キャラクターXPブースト
- コスチュームアイテム
- ラッフルチケット

**プレミアムティア報酬**（追加）:
- 限定キャラクター
- レアチャーム
- ユニークなコスチューム
- ソフト通貨乗数
- イベントテーマアイテム
- 早期レベル報酬（無料ティアを即時請求）

**イベントパス特典**:
- シーズンコンテンツとテーマ
- 期間限定の限定アイテム
- コミュニティ目標
- ボーナスチャレンジ
- コスチュームコレクション

## クロスプレイヤー競争とランキング

競争は個人のプレイを超え、コミュニティとライバル意識を育みます。

### クランランキング

クランは、ランキング対戦とトーナメントにおけるメンバーのパフォーマンスに基づいてポイントを蓄積します。

**クランポイントシステム**:
- **個々のランキング勝利**: 10～50ポイント（対戦相手の難易度に基づく）
- **トーナメントパフォーマンス**: 100～5,000ポイント（順位に基づく）
- **クランウォーズ**: 対戦勝利のボーナスポイント
- **アクティビティボーナス**: クランメンバーの参加に対するポイント
- **特別イベント**: クラン競争中のボーナスポイント

**クランリーダーボード**:
- **グローバルランキング**: 世界中のすべてのクラン
- **地域ランキング**: 地域固有の競争
- **シーズンリセット**: シーズンごとに新しいスタート
- **過去の記録**: クランの遺産を追跡

**クランランキング報酬**:
- **トップ10クラン**: 全メンバーにかなりのソフト通貨、限定クランコスチューム
- **トップ100クラン**: 中程度の報酬、クランプレステージボーナス
- **トップ1000クラン**: 少額報酬、認識バッジ
- **参加**: アクティブなすべてのクランが基本報酬を獲得

**クラン特典**:
- プレステージバッジとクランバナー
- メンバーへのソフト通貨ボーナス
- カスタマイズオプション
- トーナメントへの優先アクセス
- 限定クランイベント

### クロスリージョン トーナメント

特別なイベントでは、地域同士が友好的な競争で競い合います。

**地域競争フォーマット**:
- **地域チーム**: プレイヤーが地域を代表
- **予選**: 各地域のトッププレイヤーが選出
- **チームスコアリング**: 合計パフォーマンスで勝者が決定
- **賞品**: 地域の名誉+全参加者への報酬

**参加地域**:
- 東アジア（中国、日本、韓国）
- 東南アジア（シンガポール、タイ、ベトナム、フィリピン）
- 北米（米国、カナダ）
- ヨーロッパ（主要市場では統合または国別）
- オセアニア（オーストラリア、ニュージーランド）
- その他（成長地域）

**地域トーナメント特典**:
- 文化交流
- 地域の名誉とアイデンティティ
- 大規模な賞金プール
- 放送での露出
- 才能の発掘

<Callout type="success">
**グローバルな団結**: クロスリージョン トーナメントは、麻雀をグローバルなゲームとして祝い、健全な地域間のライバル意識を育みます。
</Callout>

### AIリーダーボード

AIコンパニオンは、人間プレイヤーとは別の独自のランキングシステムを持っています。

**AIパフォーマンス指標**:
- **勝率**: 全体的な成功率
- **平均点**: 対戦あたりの獲得点数
- **リスク管理**: 攻撃性と慎重さのバランス
- **効率性**: 意思決定の速度
- **適応性**: さまざまな対戦相手に対するパフォーマンス

**AIリーダーボードカテゴリ**:
- **全体AIランキング**: プラットフォーム全体の最もパフォーマンスの高いAI
- **バリアント固有**: 各麻雀バリアントのトップAI
- **オーナーランキング**: 最も成功したAIを持つプレイヤー
- **レンタル人気**: 最もレンタル/リクエストされたAI
- **改善率**: 最も学習速度の速いAI

**AI競争特典**:
- AIトレーニングへの投資を奨励
- AI戦略を洗練
- AIレンタルの需要を生み出す
- トレーニングの成功を示す
- 学習ベンチマークを提供する

**AIトーナメント**:
- AIのみの競技
- トレーニングされたAIの能力を披露
- 勝利したAIオーナーへの賞品
- 研究開発の洞察
- エンターテイメント価値

## 進行哲学

### バランスの取れた進歩

**デザイン原則**:
- **常に獲得できるものがある**: 複数の進行パス
- **行き止まりなし**: プレステージとシーズンシステムにより上限を防ぐ
- **スキルが最重要**: 時間だけでなくスキルによって進行が加速
- **すべての人にアクセス可能**: F2Pプレイヤーも完全に進行可能
- **オプションによる加速**: プレミアムオプションが利用可能だが必須ではない

### 長期的なエンゲージメント

**維持戦略**:
- **毎日のプレイ理由**: クエスト、ボーナス、ソーシャルアクティビティ
- **毎週の目標**: チャレンジとトーナメント
- **毎月のマイルストーン**: ロイヤルティティアとシーズンの進行
- **年間の成果**: 長期的なロイヤルティの認識
- **永続的な目標**: コレクションの完了、プレステージシステム

<Callout type="info">
**終わりのない旅**: 「Mahjong Stars」の進行システムは、カジュアルプレイヤーでも競技プレイヤーでも、常に次の目標があることを保証します。
</Callout>
