# アクセシビリティとアカウント管理

Mahjong Starsはプレイヤーを中心に設計されており、すべてのユーザーにとって簡単なアクセス、直感的なアカウント管理、柔軟なオプションを保証します。

---

## ブラウザベースのプラットフォームとデバイス互換性

Mahjong StarsはWebアプリケーションとして構築されており、ダウンロードやインストールの必要がありません。このアプローチにより、アクセシビリティと利便性が最大化されます。

### 即時アクセス

**ダウンロード不要**:
- 最新のブラウザから即座にアクセス可能
- アプリストアの承認やアップデート不要
- デバイスのストレージ容量を消費しない
- 常に最新バージョンで動作
- クロスプラットフォーム互換性

**クイックスタート**:
- 新規ユーザーは数分で登録してプレイを開始できます
- フリクションゼロのオンボーディング
- 大容量ダウンロードの待ち時間なし
- 即時ゲームプレイアクセス

<Callout type="success">
**プラットフォーム非依存**: Windows, macOS, Linux, iOS, Android、および最新のWebブラウザを搭載したあらゆるデバイスで動作します。
</Callout>

### クロスデバイスプレイ

レスポンシブデザインにより、インターフェースがさまざまな画面サイズやデバイスに適合します。

**シームレスな体験**:
- **デスクトップ/ラップトップ**: 完全な機能と最適なレイアウト
- **タブレット**: ジェスチャーサポート付きのタッチ最適化インターフェース
- **モバイル**: 小さな画面向けの合理化されたUI
- **継続的なプレイ**: あるデバイスで開始し、別のデバイスで続行
- **クラウド同期**: すべてのデバイス間で進捗が自動的に保存されます

**デバイス固有の最適化**:
- **タッチコントロール**: モバイルおよびタブレット向けの自然なジェスチャー
- **マウスとキーボード**: デスクトップでの正確なコントロール
- **ゲームパッドサポート**: オプションのコントローラー統合
- **画面回転**: 横向きおよび縦向きモード
- **スマートスケーリング**: すべての画面サイズに対応するアダプティブUI要素

### 低いハードウェア要件

重い計算はクラウドサーバーにオフロードされるため、Mahjong Starsは控えめな仕様のデバイスでもスムーズに動作します。

**最小要件**:
- **ブラウザ**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **RAM**: 最小2GB、推奨4GB
- **インターネット**: 最小2Mbps、推奨5Mbps
- **CPU**: 最新のデュアルコアプロセッサ
- **グラフィックス**: 基本的なGPUまたは統合グラフィックス

**パフォーマンスの利点**:
- 古いデバイスでもプラットフォームを実行可能
- ゲーム用ハードウェア不要
- 消費電力の削減
- エントリーバリアの低下
- より広い市場へのアクセス

<Callout type="info">
**クラウドパワー**: サーバーサイド処理により、デバイスは表示と入力のみを処理するため、プラットフォームは事実上あらゆる最新デバイスでアクセス可能です。
</Callout>

## アカウント作成とウォレット連携

Mahjong Starsのアカウント作成は簡単で、さまざまなユーザーの好みに合わせた複数のオプションがあります。

### 通常のサインアップ方法

**メールとパスワード**:
- メール認証による従来の登録
- 強力なパスワード要件
- パスワードリカバリオプション
- メール通知設定

**ソーシャルメディアログイン**:
- **Google**: クイックOAuth連携
- **Facebook**: 既存のアカウントを接続
- **Apple**: プライバシー重視のサインイン
- **Twitter/X**: ソーシャルゲーム連携
- **Discord**: コミュニティファーストのアプローチ

**ソーシャルログインの利点**:
- より高速な登録プロセス
- 覚えるパスワード不要
- 容易なアカウントリカバリ
- ソーシャルグラフ連携
- 確立されたプラットフォームを通じた信頼

### ウォレット設定（オプション）

ソフトカレンシーアカウントは自動的に作成されますが、デジタルアセットを取引したり、トークンエコシステムとやり取りしたいプレイヤーは、互換性のあるウォレットをリンクできます。

<Callout type="warning">
**完全にオプション**: Mahjong Starsのプレイにウォレット連携は必須ではありません。ブロックチェーン機能（NFT取引やトークン取引など）にのみ必要です。
</Callout>

**サポートされているウォレット**:
- **MetaMask**: 最も人気のあるEthereumウォレット
- **Trust Wallet**: モバイルフレンドリーなマルチチェーンウォレット
- **Coinbase Wallet**: 初心者向けのオプション
- **WalletConnect**: ユニバーサルウォレット接続
- **ネイティブWebウォレット**: シンプルさのための組み込みオプション

**ウォレット設定プロセス**:
1. メールまたはソーシャルログインでアカウントを作成
2. ウォレット設定に移動（オプション）
3. ウォレットプロバイダーを選択
4. 既存のウォレットを接続または新規作成
5. 接続を確認
6. ブロックチェーン機能の使用を開始

**プラットフォーム内ガイド**:
- ウォレットのステップバイステップチュートリアル
- ビデオウォークスルー
- セキュリティのベストプラクティス
- ハードウェアウォレットの推奨
- リカバリーフレーズの重要性に関する教育

### セキュリティ機能

アカウントのセキュリティはMahjong Starsで最優先事項です。

**セキュリティ対策**:
- **二要素認証（2FA）**: SMSまたは認証アプリ
- **メール認証**: アカウント操作のためのメール確認
- **ログインアラート**: 新しいデバイスログインの通知
- **セッション管理**: アクティブなセッションの表示と取り消し
- **生体認証ログイン**: モバイルでの指紋/Face ID
- **セキュリティログ**: アカウントアクティビティの追跡
- **パスワード要件**: 強力なパスワードの強制

**推奨されるベストプラクティス**:
- ユニークなパスワードを使用する
- すぐに2FAを有効にする
- アカウント認証情報を共有しない
- 大量の保有にはハードウェアウォレットを使用する
- 定期的なパスワード更新
- リカバリーフレーズの安全な保管

## 通貨オプション

Mahjong Starsは、すべてのプレイヤータイプに対応できるように設計されたデュアル通貨システムを採用しています。

### ソフト通貨（ゲーム内クレジット）

日常のゲームプレイをサポートするゲーム内クレジットで、暗号通貨の要件なしにすべてのプレイヤーがアクセスできます。

<Callout type="success">
**フリー・トゥ・アーン**: ソフト通貨は主にゲームプレイを通じて獲得されるため、Mahjong Starsは真に無料プレイ可能です。
</Callout>

**ソフト通貨の獲得方法**:
- **試合での勝利**: 主な獲得方法
- **デイリークエスト**: 目標を達成して報酬を獲得
- **ウィークリーチャレンジ**: 長期的な目標
- **実績**: マイルストーン報酬
- **トーナメント**: 競技賞
- **ログインボーナス**: アクティブプレイヤー向けの毎日の報酬
- **AI earnings**: オフライン中にAIが獲得
- **チャームボーナス**: 獲得率の向上

**ソフト通貨の使用方法**:
- メインストアからキャラクターを購入
- チャームやコスチュームを購入
- トーナメントに参加（参加費）
- AIトレーニングの加速
- スタジオのカスタマイズ
- テーマパックの購入
- ラッフルチケットの購入
- テーブルのカスタマイズ

**ソフト通貨の特性**:
- 法定通貨として直接引き出し不可
- 非投機的な設計
- 購入時の固定為替レート
- Pay-to-Winメカニズムの防止
- インフレ保護

### ハード通貨（ガバナンストークン）

プラットフォームのガバナンスやマーケットプレイス機能に参加したいプレイヤー向けの、ブロックチェーンベースのトークンです。

**トークンのユースケース**:
- **ガバナンス投票**: プラットフォームの決定に参加
- **マーケットプレイス取引**: アセットをピアツーピアで取引
- **ステーキング**: 報酬を獲得し、特典をアンロック
- **プレミアム機能**: 限定コンテンツにアクセス
- **AIマーケットプレイス**: トレーニング済みAIモデルの売買
- **トーナメント参加**: ハイステーク競技
- **アセット購入**: 特別なコレクティブル

**トークンの取得方法**:
- 取引所で購入
- 上位トーナメントで獲得
- ステーキング報酬
- ガバナンス参加報酬
- マーケットプレイス取引の利益
- 特別イベントやエアドロップ

<Callout type="info">
**別個のエンティティ**: ガバナンストークンは別個のエンティティによって発行・管理されており、Mahjong Starsは暗号通貨規制とは独立してゲームを運営できます。
</Callout>

### 入金と購入

**ソフト通貨の購入**:
- **法定通貨支払い**: クレジット/デビットカード、PayPal
- **トークン変換**: トークンをソフト通貨に変換
- **パッケージ取引**: さまざまな額面
- **ボーナスオファー**: 初回購入者ボーナス
- **安全な支払い**: PCI準拠の支払い処理

**支払いオプション**:
- クレジットカード（Visa、Mastercard、Amex）
- デビットカード
- PayPal
- Apple Pay / Google Pay
- 暗号通貨（トークン変換経由）
- 地域別支払い方法（Alipay、WeChat Payなど）

**セキュリティ**:
- 暗号化された支払い処理
- 不正検出システム
- 購入者保護ポリシー
- 返金手順
- 安全な取引履歴

### 交換と引き出し

**ソフト通貨の交換**:
ソフト通貨は主にゲーム内での使用を目的としており、法定通貨として直接引き出すことはできません。しかし、プレイヤーには間接的な交換オプションがあります。

**交換パス**:
1. ソフト通貨を使用してデジタルアセット（キャラクター、チャーム）を購入
2. アセットをNFTに変換（ブロックチェーンが有効な場合）
3. マーケットプレイスでアセットをトークンまたは法定通貨で販売
4. サードパーティの取引所を使用して最終的な変換を行う

**このモデル**:
- 潜在的な悪用を防止
- 健康的な経済を維持
- 価値抽出パスを提供する
- プレイヤーの選択と努力を必要とする
- 規制を遵守する

<Callout type="warning">
**キャッシュアウトシステムではありません**: Mahjong Starsはゲームプラットフォームであり、金融商品ではありません。アセットの価値を通じて交換は可能ですが、主な動機として設計されていません。
</Callout>

### 通貨変換

**ソフト通貨 ↔ トークン**:
- マーケットレートでトークンをソフト通貨に変換
- 一方向変換（トークン → ソフト通貨のみ）
- トークンのデフレ防止
- ソフト通貨の安定性を維持
- 明確な為替レートが表示されます

**レート決定**:
- 市場ベースの価格設定
- 透明な計算
- 隠れた手数料なし
- リアルタイム更新
- 過去のレート追跡
