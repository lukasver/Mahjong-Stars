# The Tiles Company へようこそ



## ソーシャルゲーミングの次なる進化

**The Tiles Company** は、最先端の **Web3ソーシャル麻雀P2Pゲーミングプラットフォーム** により、デジタルゲーミング空間を再定義することを目指しています。この革新的なプラットフォームは、プレイヤーが自身の戦略を学習し模倣するAIアバターを作成・トレーニングできる、革新的なAIハイブリッドシステムを活用しており、オフライン時でも **24時間年中無休** でグローバルな麻雀ゲームへの参加を可能にします。

## The Tiles Company のユニークな点は？

### 🤖 AI搭載ゲーミング
- **トレーニング可能なAIキャラクター**: あなたのプレイスタイルを学習し、自身のゲームデータでトレーニングできます。
- **24時間年中無休の継続的な流動性**: オフライン時でもAIがプレイし、制限を設定して継続的に収益を得られます。
- **ハイブリッドAIプレイ**: 自身でプレイすることと、AIに任せることをシームレスに切り替えられます。


### 🌐 Web3インテグレーション
- **Unityベースのマルチプラットフォームアプリ**: すべてのデバイスで一貫したゲーミング体験を提供します。
- **簡単なソーシャルログイン**: 組み込みのウォレット統合により、シンプルかつ安全にアクセスできます。
- **NFTマーケットプレイス**: キャラクター、コレクティブル、NFTをアンロックしてゲームプレイをパーソナライズできます。

### 🎮 ソーシャルゲーミングの卓越性
- **6億人以上のターゲット市場**: 世界中の膨大な麻雀プレイヤー層にアプローチします。
- **ソーシャルプレイ＆コミュニティ**: 友人たちとプレイし、対局をホストし、自身の麻雀サークルを成長させましょう。
- **実績のある専門知識**: 10年間の麻雀ゲーム開発経験に基づいています。
