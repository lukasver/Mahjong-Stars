## プラットフォームの理解

### 出金報酬
- **オプション機能**: アセット取引をアンロックするために出金報酬を購入する
- **サードパーティ取引**: ゲーム内アセットを外部プラットフォームで販売する
- **1:1レート**: 仮想の利益を現実世界の価値に変換する

## 初めてのゲーム

### テーブルへの参加
1. **利用可能なテーブルを閲覧**: 様々な賭け金とゲームタイプから選択する
2. **参加費の支払い**: 好みの支払い方法で参加する
3. **プレイ開始**: 現代的な機能を備えた伝統的な麻雀を楽しむ

### AIのトレーニング
1. **数ゲームプレイする**: AIはあなたのゲームプレイパターンから学習します
2. **パラメータの設定**: AIのリスク許容度と戦略を設定します
3. **パフォーマンスの監視**: AIの勝率と収益を追跡します

## 初心者向けヒント

### 🎯 ゲームプレイのヒント
- **小さく始める**: プラットフォームを学ぶために、低賭け金のテーブルから始める
- **対戦相手を研究する**: 様々なプレイスタイルと戦略を観察する
- **ソーシャル機能を使用する**: 他のプレイヤーとつながり、コミュニティから学ぶ

### 🤖 AIトレーニングのヒント
- **一貫したプレイ**: 定期的なプレイは、AIがより良く学習するのに役立ちます
- **戦略を文書化する**: AIはあなたの意思決定パターンを把握します
- **定期的な更新**: あなたが上達するにつれて、AIのトレーニングを続けます

### 💰 経済のヒント
- **予算管理**: ゲーム活動の支出限度額を設定します
- **パフォーマンスの追跡**: 勝ち負けを監視します
- **手数料の理解**: すべての取引コストを認識します
