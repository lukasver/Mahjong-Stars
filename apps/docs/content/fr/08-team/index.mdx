# L'équipe Mahjong Stars

Notre équipe de classe mondiale combine des décennies d'expertise en Mahjong, d'expérience dans l'industrie du jeu et d'innovation technologique, positionnant Mahjong Stars pour un succès mondial dans le paysage évolutif des jeux sociaux.
