# 可访问性与账户管理

《麻将之星》（Mahjong Stars）以玩家为中心进行设计，确保所有用户都能轻松访问、直观地管理账户，并享有灵活的选项。

---

## 基于浏览器的平台和设备兼容性

《麻将之星》是一款网络应用程序，无需下载或安装。这种方法最大限度地提高了可访问性和便利性。

### 即时访问

**无需下载**:
- 可从任何现代浏览器即时访问
- 无需应用商店审批或更新
- 不占用设备存储空间
- 始终运行最新版本
- 跨平台兼容

**快速开始**:
- 新用户可在几分钟内注册并开始游戏
- 零摩擦入门流程
- 无需等待大型下载
- 即时游戏访问

<Callout type="success">
**平台无关**: 适用于 Windows、macOS、Linux、iOS、Android 以及任何拥有现代网络浏览器的设备。
</Callout>

### 跨设备游玩

响应式设计确保界面可适应不同的屏幕尺寸和设备。

**无缝体验**:
- **台式机/笔记本电脑**: 功能齐全的体验，布局优化
- **平板电脑**: 触摸优化界面，支持手势
- **移动设备**: 针对小屏幕精简的用户界面
- **连续游玩**: 在一台设备上开始，在另一台设备上继续
- **云同步**: 进度自动在所有设备上保存

**特定设备优化**:
- **触摸控件**: 移动设备和平板电脑上的自然手势
- **鼠标和键盘**: 台式机上的精确控制
- **游戏手柄支持**: 可选控制器集成
- **屏幕旋转**: 横向和纵向模式
- **智能缩放**: 适用于所有屏幕尺寸的自适应 UI 元素

### 低硬件要求

由于繁重的计算已卸载到云服务器，《麻将之星》可在配置一般的设备上流畅运行。

**最低要求**:
- **浏览器**: Chrome 90+、Firefox 88+、Safari 14+、Edge 90+
- **内存**: 最低 2GB，推荐 4GB
- **网络**: 最低 2 Mbps，推荐 5 Mbps
- **CPU**: 任何现代双核处理器
- **图形**: 基本 GPU 或集成显卡

**性能优势**:
- 旧设备也可运行平台
- 无需游戏硬件
- 降低功耗
- 降低入门门槛
- 更广泛的市场可访问性

<Callout type="info">
**云驱动**: 服务器端处理意味着您的设备只需处理显示和输入，使平台几乎适用于任何现代设备。
</Callout>

## 账户创建与钱包集成

在《麻将之星》上创建账户非常简单，提供多种选项以满足不同的用户偏好。

### 标准注册方法

**电子邮件和密码**:
- 传统的电子邮件验证注册
- 强密码要求
- 密码恢复选项
- 电子邮件通知偏好

**社交媒体登录**:
- **Google**: 快速 OAuth 集成
- **Facebook**: 连接现有账户
- **Apple**: 注重隐私的登录
- **Twitter/X**: 社交游戏集成
- **Discord**: 以社区为先的方法

**社交登录优势**:
- 更快的注册过程
- 无需记住密码
- 更轻松的账户恢复
- 社交图谱集成
- 通过成熟平台建立信任

### 钱包设置（可选）

尽管游戏内货币账户会自动创建，但希望交易数字资产或与代币生态系统互动 の玩家可以链接兼容的钱包。

<Callout type="warning">
**完全可选**: 钱包集成不是玩《麻将之星》所必需的。它仅用于区块链功能，如 NFT 交易和代币交易。
</Callout>

**支持的钱包**:
- **MetaMask**: 最流行的以太坊钱包
- **Trust Wallet**: 移动友好型多链钱包
- **Coinbase Wallet**: 对初学者友好的选项
- **WalletConnect**: 通用钱包连接
- **原生网页钱包**: 内置选项，简单易用

**钱包设置流程**:
1. 使用电子邮件或社交登录创建账户
2. 导航到钱包设置（可选）
3. 选择钱包提供商
4. 连接现有钱包或创建新钱包
5. 验证连接
6. 开始使用区块链功能

**平台内指南**:
- 分步钱包教程
- 视频演练
- 安全最佳实践
- 硬件钱包推荐
- 恢复短语重要性教育

### 安全功能

账户安全在《麻将之星》中至关重要。

**安全措施**:
- **双重身份验证 (2FA)**: 短信或身份验证器应用程序
- **电子邮件验证**: 确认账户操作的电子邮件
- **登录提醒**: 新设备登录通知
- **会话管理**: 查看和撤销活动会话
- **生物识别登录**: 移动设备上的指纹/面部 ID
- **安全日志**: 跟踪账户活动
- **密码要求**: 强制执行强密码

**鼓励的最佳实践**:
- 使用唯一密码
- 立即启用 2FA
- 不要共享账户凭据
- 为大量持有资产使用硬件钱包
- 定期更新密码
- 安全存储恢复短语

## 货币选项

《麻将之星》采用双重货币系统，旨在满足所有类型的玩家。

### 游戏内货币（游戏内积分）

用于日常游戏的游戏内积分，所有玩家均可使用，无需加密货币。

<Callout type="success">
**免费赚取**: 游戏内货币主要通过游戏赚取，使《麻将之星》真正免费游玩。
</Callout>

**赚取游戏内货币**:
- **赢得比赛**: 主要赚取方式
- **每日任务**: 完成目标以获得奖励
- **每周挑战**: 长期目标
- **成就**: 里程碑奖励
- **锦标赛**: 比赛奖品
- **登录奖励**: 活跃玩家的每日奖励
- **AI 收益**: 您离线时 AI 也会为您赚钱
- **魅力奖励**: 提高赚取率

**使用游戏内货币**:
- 从主商店购买角色
- 购买魅力和装饰品
- 参加锦标赛（入门费）
- 加速 AI 训练
- 自定义工作室
- 购买主题包
- 购买抽奖券
- 桌子自定义

**游戏内货币属性**:
- 不能直接兑换为法定货币
- 非投机设计
- 固定购买汇率
- 防止“付费获胜”机制
- 防通胀

### 硬通货（治理代币）

供希望参与平台治理和市场功能的玩家使用的基于区块链的代币。

**代币用途**:
- **治理投票**: 参与平台决策
- **市场交易**: 点对点交易资产
- **质押**: 赚取奖励并解锁福利
- **高级功能**: 访问独家内容
- **AI 市场**: 买卖训练有素的 AI 模型
- **锦标赛报名**: 高风险比赛
- **资产购买**: 特殊收藏品

**获取代币**:
- 在交易所购买
- 通过高级锦标赛赚取
- 质押奖励
- 治理参与奖励
- 市场交易利润
- 特殊活动和空投

<Callout type="info">
**独立实体**: 治理代币由独立实体发行和管理，使《麻将之星》能够独立于加密货币法规运营游戏。
</Callout>

### 充值和购买

**购买游戏内货币**:
- **法定货币支付**: 信用卡/借记卡、PayPal
- **代币兑换**: 兑换代币以获得游戏内货币
- **套餐**: 提供各种面额
- **奖励优惠**: 首购奖励
- **安全支付**: 符合 PCI 标准的支付处理

**支付选项**:
- 信用卡（Visa、Mastercard、Amex）
- 借记卡
- PayPal
- Apple Pay / Google Pay
- 加密货币（通过代币兑换）
- 区域支付方式（支付宝、微信支付等）

**安全**:
- 加密支付处理
- 欺诈检测系统
- 买家保护政策
- 退款流程
- 安全的交易历史

### 兑换和提现

**游戏内货币兑换**:
游戏内货币主要用于游戏内使用，不能直接兑换为法定货币。但是，玩家有间接兑换选项：

**兑换路径**:
1. 使用游戏内货币购买数字资产（角色、魅力）
2. 将资产转换为 NFT（如果启用了区块链）
3. 在市场上将资产出售以换取代币或法定货币
4. 使用第三方交易所进行最终兑换

**此模式**:
- 防止潜在滥用
- 维持健康的经济
- 提供价值提取路径
- 需要玩家的选择和努力
- 符合法规

<Callout type="warning">
**非套现系统**: 《麻将之星》是一个游戏平台，而不是金融工具。可以通过资产价值进行兑换，但并非主要动机。
</Callout>

### 货币兑换

**游戏内货币 ↔ 代币**:
- 以市场价格将代币兑换为游戏内货币
- 单向兑换（仅限代币 → 游戏内货币）
- 防止代币通缩
- 保持游戏内货币稳定
- 显示清晰的汇率

**汇率确定**:
- 基于市场的定价
- 透明的计算
- 无隐藏费用
- 实时更新
- 历史汇率跟踪
