# 玩家成长与留存

成长是长期参与度的重要组成部分。《Mahjong Stars》融入了多种升级系统、奖励机制和竞技结构，以激励并保持玩家的投入。

---

## 等级系统与AI成长

《Mahjong Stars》拥有相互关联的成长系统，提供持续的进步机会。

### 玩家等级

反映整体游戏经验，并在玩家进步时解锁新功能。

**等级成长**:
- **起始等级**: 1（所有玩家从这里开始）
- **最高等级**: 100（有扩展空间）
- **经验来源**: 进行的比赛、完成的任务、获得的成就
- **等级要求**: 指数级增长的所需经验值

**经验值（XP）获取**:
- **完成比赛**: 50-200 XP（根据表现）
- **胜利**: 额外奖励 100-300 XP
- **完美对局**: 额外奖励 500 XP
- **每日任务**: 每个 200-1,000 XP
- **每周挑战**: 1,000-5,000 XP
- **成就**: 500-10,000 XP
- **锦标赛参与**: 1,000-50,000 XP

**按等级解锁的功能**:

| 等级范围 | 解锁的功能 |
|------------|-------------------|
| 1-10 | 基本游戏、教程、休闲比赛 |
| 11-20 | 排位赛、第一个AI槽位、每日任务 |
| 21-30 | 高级教程、锦标赛准入、加入帮派 |
| 31-40 | 工作室准入（如果拥有）、AI租赁能力、自定义房间 |
| 41-50 | 高级锦标赛、创建帮派、市场出售 |
| 51-60 | 高风险牌桌、高级功能、指导计划 |
| 61-70 | 专属锦标赛、早期功能访问 |
| 71-80 | 威望选项、传奇物品访问 |
| 81-90 | 精英竞赛、平台测试计划 |
| 91-100 | 最大化收益、特殊认可、独家内容 |

**升级奖励**:
- 软货币奖励
- 魅力奖励
- 角色碎片
- 彩票
- 装饰品解锁
- 称号选项

<Callout type="success">
**持续成长**: 即使是满级玩家，也能通过威望系统和赛季重置继续获得奖励。
</Callout>

### 角色等级

每个角色通过比赛积累经验，解锁额外的功能和奖励。

**角色成长**:
- **单独经验值**: 每个角色单独升级
- **活跃角色获得经验**: 仅装备的角色
- **最高等级**: 每个角色50级
- **威望选项**: 重置为1级，获得永久奖励

**角色经验值来源**:
- 在比赛中使用角色（主要）
- 使用角色获胜（奖励）
- 特殊角色挑战
- 角色专属成就
- 活动参与

**升级福利**:
- **装饰选项**: 在特定等级解锁
- **表情**: 角色专属表情（5、15、25、35、45级）
- **AI效率**: 提高AI训练速度（每级+2%）
- **属性加成**: 次要属性提升
- **技能点**: 分配到技能树（每级1点）

**技能树分配**:

**策略分支**:
- AI训练速度加成
- 额外AI槽位解锁（20、40级）
- 增强赛后分析
- 策略提示改进
- 概率计算辅助

**社交分支**:
- 独特表情和贴纸
- 聊天自定义选项
- 好友奖励倍增器
- 帮派贡献增加
- 社交经验值加成

**经济分支**:
- 软货币赚取加成（每点+1%）
- 彩票倍增器
- 锦标赛入场费折扣（最高25%）
- 市场交易费降低
- 魅力掉落率增加

**角色威望**:
- 50级可用
- 重置为1级，但保留技能点
- 获得永久属性加成
- 可视化威望指示器
- 独家装饰选项
- 可进行多次威望升级

### AI等级

随着AI伙伴进行更多比赛，它们会获得等级，从而提高性能。

**AI成长机制**:
- **数据驱动升级**: 通过更多游戏数据提高
- **比赛经验**: 每场比赛都有助于AI成长
- **最高等级**: 每个AI实例50级
- **品质等级**: 青铜、白银、黄金、白金、钻石

**AI等级福利**:
- **更快的决策**: 思考时间缩短（快5-50%）
- **更好的风险评估**: 提高概率计算能力
- **增强适应性**: 更快适应对手
- **策略优化**: 更复杂的玩法模式
- **一致性**: 更可靠的表现

**AI训练要求**:
- **初始训练**: 50-100场比赛以获得基本功能
- **熟练度**: 200-500场比赛以获得可靠表现
- **优秀**: 1,000+场比赛以获得顶级AI
- **维护**: 持续游戏以保持AI更新
- **再训练**: 使用新策略更新AI

**AI训练加速**:
- **工作室训练设施**: 更快的训练
- **训练模块**: 可购买的XP提升
- **模拟比赛**: 计入AI经验
- **定向训练**: 专注于特定弱点
- **批量训练**: 同时训练多个AI（工作室功能）

<Callout type="info">
**鲜活的AI**: 您的AI会随着您的游戏不断进化，适应元变化，并根据累积的经验优化策略。
</Callout>

## 奖励与忠诚度计划

为激励频繁游戏，《Mahjong Stars》提供全面的奖励系统。

### 每日任务

通过正常游戏即可完成的简单目标。

**任务类型**:
- **获胜X场比赛**: 赢得3场比赛（奖励：1,000软货币）
- **进行Y场游戏**: 完成5场游戏，无论结果如何（奖励：500软货币）
- **使用特定角色**: 使用特色角色玩3场比赛（奖励：角色XP加成）
- **社交任务**: 与朋友一起玩，发送礼物（奖励：社交物品）
- **变体任务**: 玩特定的麻将变体（奖励：变体专属奖励）

**任务系统**:
- **每日重置**: 每24小时刷新新任务
- **任务槽位**: 同时拥有3-5个活跃任务
- **重置选项**: 每天可替换一个任务（需消耗软货币）
- **连胜奖励**: 连续多天游戏可倍增奖励
- **快速完成**: 大多数任务可在30-60分钟内完成

### 每周挑战

需要战略规划的长期目标。

**挑战示例**:
- **获得特定牌型胜利**: 达成特定的赢牌模式（3次）
- **达到段位里程碑**: 提升排位阶梯（奖励：段位奖励）
- **锦标赛表现**: 在每周锦标赛中排名前50
- **收集目标**: 获得5个新魅力
- **AI训练**: 使用AI进行50场比赛（奖励：AI XP加成）

**每周挑战福利**:
- **丰厚奖励**: 魅力礼包、角色碎片、大量软货币
- **策略深度**: 需要规划和奉献
- **成长焦点**: 鼓励特定的游戏风格
- **社区活动**: 有时与全球目标挂钩

### 忠诚度等级

玩家根据连续游戏天数和总比赛场次累积忠诚度积分。

**忠诚度等级系统**:

| 等级 | 要求 | 福利 |
|------|-------------|----------|
| **青铜** | 连续7天，共50场比赛 | +5%软货币，1张每周彩票 |
| **白银** | 连续30天，共200场比赛 | +10%软货币，2张每周彩票，锦标赛折扣（10%） |
| **黄金** | 连续90天，共500场比赛 | +15%软货币，3张每周彩票，锦标赛折扣（15%），独家魅力 |
| **白金** | 连续180天，共1,000场比赛 | +20%软货币，5张每周彩票，锦标赛折扣（20%），优先支持 |
| **钻石** | 连续365天，共2,500场比赛 | +25%软货币，10张每周彩票，锦标赛折扣（25%），专属访问 |

**忠诚度福利**:
- **软货币倍增器**: 永久性赚取增加
- **费用降低**: 锦标赛入场费折扣
- **彩票访问**: 自动获得独家彩票资格
- **优先功能**: 新内容的提前访问
- **装饰奖励**: 忠诚度专属物品
- **身份认可**: 特殊徽章和称号

**保持忠诚度**:
- 每日登录即可（无需比赛）
- 允许漏掉几天的宽限期（较高等级有2天缓冲）
- 假期模式（暂停忠诚度，黄金级以上可用）
- 如果连胜中断，提供回归奖励

<Callout type="warning">
**连胜保护**: 较高的忠诚度等级提供漏掉日期的保护，但保持一致性是获得最大福利的关键。
</Callout>

### 活动通行证

赛季活动引入带有逐步奖励的活动通行证。

**活动通行证结构**:
- **免费等级**: 所有玩家均可获得，奖励适中
- **高级等级**: 可选购，奖励更丰厚
- **持续时间**: 通常每赛季4-8周
- **等级**: 每张通行证50-100级

**赚取活动XP**:
- 玩比赛（任何模式）
- 完成活动专属任务
- 参加主题锦标赛
- 每日登录奖励
- 特殊活动挑战

**免费等级奖励**（示例）：
- 软货币礼包
- 普通魅力
- 角色XP加成
- 装饰品
- 彩票

**高级等级奖励**（额外）：
- 独家角色
- 稀有魅力
- 独特装饰品
- 软货币倍增器
- 活动主题物品
- 提前等级奖励（立即领取免费等级）

**活动通行证福利**:
- 季节性内容和主题
- 限时独家物品
- 社区目标
- 奖励挑战
- 装饰品收藏

## 跨玩家竞赛与排名

竞赛超越个人游戏，培养社区和竞争。

### 帮派排名

帮派根据成员在排位赛和锦标赛中的表现累积积分。

**帮派积分系统**:
- **个人排位赛胜利**: 10-50积分（根据对手难度）
- **锦标赛表现**: 100-5,000积分（根据排名）
- **帮派战争**: 主动对决胜利可获得额外积分
- **活跃奖励**: 帮派成员参与可获得积分
- **特殊活动**: 帮派竞赛期间有额外积分

**帮派排行榜**:
- **全球排名**: 全世界所有帮派
- **区域排名**: 特定地区的竞争
- **赛季重置**: 每赛季重新开始
- **历史记录**: 记录帮派的辉煌

**帮派排名奖励**:
- **前10名帮派**: 所有成员获得大量软货币，独家帮派装饰品
- **前100名帮派**: 中等奖励，帮派威望奖励
- **前1000名帮派**: 小额奖励，认可徽章
- **参与奖励**: 所有活跃帮派获得基础奖励

**帮派福利**:
- 威望徽章和帮派旗帜
- 成员获得软货币加成
- 自定义选项
- 优先锦标赛访问
- 独家帮派活动

### 跨区域锦标赛

特殊活动将各个地区进行友好竞赛。

**区域竞赛形式**:
- **区域队伍**: 玩家代表各自地区
- **资格赛**: 选拔各地区顶尖玩家
- **团队积分**: 综合表现决定胜者
- **奖品**: 地区荣誉+为所有参赛者准备的奖励

**参赛地区**:
- 东亚（中国、日本、韩国）
- 东南亚（新加坡、泰国、越南、菲律宾）
- 北美（美国、加拿大）
- 欧洲（合并或按主要市场国家划分）
- 大洋洲（澳大利亚、新西兰）
- 其他（正在增长的地区）

**区域锦标赛福利**:
- 文化交流
- 地区荣誉和身份认同
- 巨额奖金池
- 广播可见性
- 人才发掘

<Callout type="success">
**全球团结**: 跨区域锦标赛颂扬麻将作为一项全球性游戏，同时促进健康的区域竞争。
</Callout>

### AI排行榜

AI伙伴拥有自己的排名系统，与人类玩家分开。

**AI表现指标**:
- **胜率**: 整体成功率
- **平均得分**: 每场比赛获得的得分
- **风险管理**: 侵略性和谨慎性的平衡
- **效率**: 决策速度
- **适应性**: 在不同对手下的表现

**AI排行榜类别**:
- **整体AI排名**: 平台内表现最佳的AI
- **特定变体**: 每个麻将变体的顶尖AI
- **玩家排名**: 拥有最成功AI的玩家
- **租赁人气**: 最受欢迎/最常被租用的AI
- **进步率**: 学习速度最快的AI

**AI竞赛福利**:
- 鼓励在AI训练上的投资
- 优化AI策略
- 创造AI租赁需求
- 展示训练成果
- 提供学习基准

**AI锦标赛**:
- 仅限AI参与的竞赛
- 展示训练有素的AI能力
- 为获胜AI所有者提供奖品
- 提供研发见解
- 娱乐价值

## 成长理念

### 平衡的进步

**设计原则**:
- **总有可赚取的东西**: 多种成长路径
- **无死胡同**: 威望和赛季系统防止封顶
- **技巧至关重要**: 成长速度与技巧相关，而不仅仅是时间
- **人人可及**: 免费玩家也能完全成长
- **可选加速**: 提供高级选项，但非必需

### 长期参与

**留存策略**:
- **每日游玩理由**: 任务、奖励、社交活动
- **每周目标**: 挑战和锦标赛
- **每月里程碑**: 忠诚度等级和赛季进展
- **年度成就**: 长期忠诚度认可
- **持续目标**: 收藏品完成、威望系统

<Callout type="info">
**无尽的旅程**: 《Mahjong Stars》的成长系统确保总有下一个目标，无论您是休闲玩家还是竞技玩家。
</Callout>
