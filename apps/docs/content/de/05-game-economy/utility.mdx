# Nutzen & Erwerb

Der Governance-Token hat über die einfache Zahlung hinaus mehrere Nutzungsfunktionen, die eine nachhaltige Token-Ökonomie mit echten Werttreibern schaffen.

---

## Nutzen und Anwendungsfälle

Der Governance-Token dient verschiedenen Zwecken im Mahjong Stars-Ökosystem:

### VIP-Funktionen

Für bestimmte Premium-Funktionen ist möglicherweise das Halten oder Staking einer Mindestanzahl von Tokens erforderlich.

**Token-geschützte Funktionen**:
- **Priorisierte Spielersuche**: Reduzierte Wartezeiten
- **Früher Charakterzugang**: Erster Zugriff auf neue Veröffentlichungen
- **Verbesserte Charm-Drops**: Erhöhte Drop-Raten (10-25%)
- **Premium-Analysen**: Fortgeschrittene statistische Werkzeuge
- **Benutzerdefinierte Tische**: Zusätzliche Anpassungsoptionen
- **Speichererweiterung**: Mehr Replay-Speicher, Charakter-Slots

**Halte-Stufen**:
- **Bronze Holder** (1.000 Tokens): Grundlegende VIP-Funktionen
- **Silver Holder** (5.000 Tokens): Erweiterte Vorteile
- **Gold Holder** (10.000 Tokens): Premium-Zugang
- **Platinum Holder** (25.000 Tokens): Elite-Status
- **Diamond Holder** (50.000+ Tokens): Maximale Vorteile

<Callout type="info">
**Halten zum Freischalten**: Allein das Halten von Tokens in Ihrer Wallet schaltet Vorteile frei. Für grundlegende VIP-Funktionen ist kein Staking erforderlich (obwohl Staking zusätzliche Vorteile bietet).
</Callout>

### Exklusive Turniere

Hochkarätige Turniere mit großen Preispools erfordern möglicherweise Token-Eintrittsgebühren.

**Token-Turnierarten**:

**Wöchentlicher High Roller**:
- Teilnahmegebühr: 100-500 Tokens
- Preispool: 50.000-250.000 Tokens
- Format: Schweizer System, 64-256 Spieler
- Belohnungen: Top-10-Platzierungen

**Monatsmeisterschaft**:
- Teilnahmegebühr: 1.000 Tokens
- Preispool: 500.000+ Tokens + seltene Assets
- Format: Mehrtägige Qualifikation + Finale
- Belohnungen: Top-20-Platzierungen + besondere Anerkennung

**Saisonübergreifender Grand Prix**:
- Teilnahmegebühr: Qualifikationsbasiert (keine direkte Token-Gebühr)
- Preispool: 2.000.000+ Tokens + legendäre Gegenstände
- Format: Regionale Qualifikationen → Globale Finals
- Belohnungen: Top-50 + Teilnahmeboni

**Vorteile**:
- Höherer Wettbewerb der Fähigkeiten
- Größere Preispools
- Professionelle Anerkennung
- Weg zu E-Sport-Karrieren
- Exklusive Titel und Kosmetika

### Rabatte und Rückerstattungen

Das Staking von Tokens kann die Marktplatz-Kommissionsgebühren reduzieren oder Rückerstattungen von Soft-Währung für Einkäufe gewähren.

**Staking-Vorteile**:

| Staking-Stufe | Marktplatz-Gebührsrabatt | Soft-Währungs-Rückerstattung |
|-------------|-------------------------|---------------------|
| **Kein Stake** | 0 % (10 % Basisgebühr) | 0 % |
| **Bronze** (1K gestaket) | 10 % Rabatt (9 % Gebühr) | 5 % Rückerstattung auf Einkäufe |
| **Silver** (5K gestaket) | 25 % Rabatt (7,5 % Gebühr) | 10 % Rückerstattung |
| **Gold** (10K gestaket) | 40 % Rabatt (6 % Gebühr) | 15 % Rückerstattung |
| **Platinum** (25K gestaket) | 50 % Rabatt (5 % Gebühr) | 20 % Rückerstattung |
| **Diamond** (50K+ gestaket) | 60 % Rabatt (4 % Gebühr) | 25 % Rückerstattung |

**Rückerstattungsmechanismus**:
- Automatische Rückerstattung von Soft-Währung für Einkäufe
- Angewendet zum Zeitpunkt der Transaktion
- Kombinierbar mit anderen Boni
- Keine Obergrenzen für die Gesamtrückerstattung
- Fördert das Engagement auf der Plattform

<Callout type="success">
**Echte Ersparnisse**: Für aktive Händler und Spieler können Gebührenrabatte und Rückerstattungen im Laufe der Zeit erhebliche Vorteile bieten.
</Callout>

### KI-Marktplatz

Spieler können KI-Modelle auf einem KI-Marktplatz entwickeln und verkaufen.

**KI-Marktplatz-Funktionen**:

**Verkauf von KI-Modellen**:
- **Trainierte KI-Vorlagen**: Verkaufen Sie Ihre erfolgreichen KI-Strategien
- **Spezialisierte Module**: Erstellen Sie KI-Pakete für Varianten
- **Trainingsdatensätze**: Verkaufen Sie anonymisierte Trainingsdaten
- **Strategieführer**: KI-gestützte Strategiedokumentation
- **Beratungsdienste**: Eins-zu-eins-KI-Trainingsunterstützung

**Preisgestaltung**:
- Verkäufer legen ihre eigenen Preise fest (in Tokens)
- Die Plattform erhebt eine Provision von 5-15 % (reduziert durch Staking)
- Käufer erhalten sofortigen Zugriff
- Bewertungen und Rezensionen
- Beliebte KIs können erhebliche Einnahmen generieren

**KI als geistiges Eigentum**:
- Schützen Sie Ihre KI-Strategien
- Monetarisieren Sie Ihre Trainingsinvestitionen
- Bauen Sie sich einen Ruf als KI-Trainer auf
- Schaffen Sie passive Einkommensströme
- Tragen Sie zum Plattform-Ökosystem bei

**Kauf von KI-Modellen**:
- Kaufen Sie bewährte Strategien
- Überspringen Sie die Trainingszeit
- Lernen Sie von Experten
- Experimentieren Sie mit verschiedenen Ansätzen
- Beschleunigen Sie die Wettbewerbsleistung

## Erwerb und Zirkulation

Um eine gesunde Token-Ökonomie aufrechtzuerhalten, setzt Mahjong Stars Mechanismen für eine kontrollierte Token-Versorgung und eine faire Verteilung ein.

### Anfängliche Verteilung

Tokens werden bei der Markteinführung verschiedenen Interessengruppen zugewiesen:

**Aufschlüsselung der Verteilung** (aus dem Whitepaper):
- **Pre-Seed**: 3,28 %
- **Seed**: 6,09 %
- **Privat**: 9,38 %
- **KOL (Key Opinion Leaders)**: 2,25 %
- **Öffentlicher Verkauf**: 3,75 %
- **Liquidität**: 9,38 %
- **Team**: 12,00 %
- **Beratung**: 0,81 %
- **Partnerschaften**: 2,19 %
- **Reserve**: 7,50 %
- **F&E**: 2,06 %
- **Betriebskosten**: 2,20 %
- **Airdrop**: 3,78 %
- **In-Game-Belohnungen**: 33,33 % (größte Zuteilung)
- **Marketing**: 2,00 %

<Callout type="warning">
**Spieler-fokussiert**: Beachten Sie, dass In-Game-Belohnungen (33,33 %) die größte Einzelzuteilung darstellen, was das Engagement für den Spielwert unterstreicht.
</Callout>

### Verdienstmechanismen

Tokens können durch Gameplay und Teilnahme an der Plattform verdient werden:

**Verdienen durch Gameplay**:
- **Turnierpreise**: Top-Platzierungen in Token-Turnieren
- **Saisonale Meisterschaften**: Belohnungen am Ende der Saison
- **Erfolgsboni**: Seltene Errungenschaften (50-5.000 Tokens)
- **Bestenlisten-Belohnungen**: Spieler auf Spitzenplätzen (wöchentlich, monatlich)
- **Perfekte Spiele**: Boni für außergewöhnliche Leistungen

**Teilnahmebelohnungen**:
- **Staking-Renditen**: Passives Verdienen durch Staking
- **Governance-Teilnahme**: Abstimmung über Vorschläge (10-100 Tokens pro Stimme)
- **Marktplatz-Handel**: Profitieren Sie von erfolgreichen Trades
- **KI-Marktplatz**: Verkaufen Sie KI-Modelle und -Dienste
- **Inhaltserstellung**: Belohnung für Community-Beiträge

**Sonderveranstaltungen**:
- **Saisonale Veranstaltungen**: Zeitlich begrenzte Verdienstmöglichkeiten
- **Community-Meilensteine**: Belohnungen für das Erreichen von Plattform-weiten Zielen
- **Partnerschaftsveranstaltungen**: Kooperationen mit anderen Projekten
- **Empfehlungsboni**: Bringen Sie Freunde mit, verdienen Sie Tokens
- **Bug-Bounties**: Melden Sie Probleme, erhalten Sie Belohnungen

### Airdrops

Strategische Airdrops belohnen loyale Community-Mitglieder und ziehen neue Nutzer an:

**Airdrop-Kategorien**:

**Loyalitäts-Airdrops**:
- Belohnen bestehende Spieler
- Basierend auf Aktivitätsniveau
- Vierteljährliche Verteilung
- Zuteilung: 1 % des Gesamtangebots pro Jahr

**Partnerschafts-Airdrops**:
- Cross-Promotion-Kampagnen
- Kooperationen mit Spieleplattformen
- Cross-Pollination der Community
- Aufbau strategischer Ökosysteme

**Event-Airdrops**:
- Turnier-Zuschauer
- Social-Media-Teilnehmer
- Content Creators
- Community-Helfer

**Teilnahmeberechtigung**:
- Anforderungen an das Account-Alter
- Mindestaktivitätsniveaus
- KYC-Abschluss (für größere Airdrops)
- Keine Bot-Konten
- Guter Leumund (keine Verstöße)

### In-Game-Token-Integration

Tokens können im Spiel für bestimmte Funktionen verwendet werden:

**Turnierteilnahme**:
- Erforderlich für Premium-Turniere
- Optional für erhöhte Preispools
- Rückerstattung bei Turnierabbruch
- Wiederverteilung in den Preispool

**Spezielle Spielmodi**:
- Token-gestakte Spiele (der Gewinner erhält alles)
- High-Roller-Tische
- Profi-Ligen
- Einladungs-Events

**Kauf von Assets**:
- Exklusive limitierte Charaktere
- Legendäre Charms
- Premium-Studios
- Spezielle Kooperationen

<Callout type="info">
**Ausgewogene Integration**: Die Token-Nutzung im Gameplay ist immer optional, sodass auch Nicht-Token-Inhaber die Plattform vollständig genießen können.
</Callout>

## Zirkulation und Angebotsmanagement

### Kontrolliertes Angebot

**Festes maximales Angebot**:
- Gesamtangebot begrenzt (verhindert Inflation)
- Keine zusätzliche Prägung (nach anfänglicher Verteilung)
- Transparente Zuteilung
- Vorhersehbarer Veröffentlichungsplan

**Vesting-Zeitpläne**:
- Team-Tokens: 4 Jahre Vesting mit 1 Jahr Cliff
- Berater-Tokens: 2 Jahre Vesting
- Strategische Partner: Benutzerdefiniertes Vesting
- Frühinvestoren: 1-2 Jahre Vesting
- Community-Belohnungen: Schrittweise über 5 Jahre freigegeben

**Veröffentlichungsplan**:
- Verhindert plötzliche Angebotsschocks
- Richtet Anreize langfristig aus
- Aufrechterhaltung der Preisstabilität
- Baut Vertrauen bei den Inhabern auf

### Burn-Mechanismen

Um deflationären Druck zu erzeugen und den Token-Wert zu erhalten:

**Transaktionsgebühren**:
- 1-2 % der Marktplatzgebühren werden verbrannt
- Permanente Entfernung aus der Zirkulation
- Automatisierter Burn-Prozess
- Transparente Nachverfolgung

**Asset-Käufe**:
- Ein Teil der primären Asset-Verkäufe wird verbrannt
- Reduziert das Angebot bei gleichzeitiger Steigerung des Nutzens
- Vorteile für alle Token-Inhaber
- Regelmäßige Burn-Berichte

**Turniergebühren**:
- Ein kleiner Prozentsatz der Teilnahmegebühren wird verbrannt
- Erhöht die Token-Knappheit im Laufe der Zeit
- Nachhaltiges deflationäres Modell

**Burn-Transparenz**:
- Öffentliche Burn-Wallet-Adresse
- Regelmäßige Burn-Ankündigungen
- Tools zur Community-Nachverfolgung
- Vierteljährliche Burn-Berichte

### Liquiditätsmanagement

**Börsennotierungen**:
- Wichtige zentrale Börsen (CEX)
- Dezentrale Börsen (DEX)
- Liquiditätspaarungen (ETH, USDT usw.)
- Tiefe Liquidität für minimale Slippage

**Liquiditätsanreize**:
- Belohnungen für Liquiditätsanbieter
- Reduzierte Volatilität durch Tiefe
- Anteil an Handelsgebühren
- Langfristige LP-Boni

**Market Making**:
- Professionelle Market Maker
- Konsistente Geld-/Briefspannen
- Aufrechterhaltung des Volumens
- Unterstützung der Preisentdeckung

## Ökonomische Nachhaltigkeit

### Einnahmen → Rückkäufe

Ein Teil der Plattform-Einnahmen wird für den Rückkauf von Tokens verwendet:

**Rückkaufprogramm**:
- Vierteljährliche Rückkaufveranstaltungen
- Nutzung von Plattformgewinnen
- Unterstützung des Token-Preises
- Reduzierung des zirkulierenden Angebots
- Langfristiger Vorteil für Inhaber

**Transparenz**:
- Vorab-Ankündigungen
- Öffentliche Wallet-Adressen
- Detaillierte Berichte
- Aufsicht durch die Community

### Schatzkammer-Management

Die Schatzkammer der Plattform wird professionell verwaltet:

**Funktionen der Schatzkammer**:
- Notfallreserven
- Finanzierung der Entwicklung
- Marketingkampagnen
- Investitionen in Partnerschaften
- Community-Initiativen

**Governance**:
- Zustimmung der Token-Inhaber für größere Ausgaben
- Transparente Buchführung
- Regelmäßige Audits
- Sichtbarkeit für die Community
