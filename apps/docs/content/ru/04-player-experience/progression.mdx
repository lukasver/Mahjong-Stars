# Прогресс и Удержание Игроков

Прогресс — неотъемлемый компонент долгосрочного вовлечения. Mahjong Stars использует множество систем уровней, механизмов вознаграждения и соревновательных структур, чтобы поддерживать мотивацию и вовлеченность игроков.

---

## Системы Уровней и Развитие ИИ

Mahjong Stars имеет взаимосвязанные системы прогрессии, которые предоставляют непрерывные возможности для развития.

### Уровень Игрока

Отражает общий опыт и открывает новые функции по мере продвижения игроков.

**Прогрессия Уровня**:
- **Начальный Уровень**: 1 (все игроки начинают здесь)
- **Максимальный Уровень**: 100 (с возможностью расширения)
- **Источники Опыта**: Сыгранные матчи, выполненные квесты, полученные достижения
- **Требования к Уровню**: Экспоненциально увеличивающееся количество требуемого опыта

**Получение Опыта (XP)**:
- **Завершение Матча**: 50-200 XP (в зависимости от производительности)
- **Победы**: Бонус 100-300 XP
- **Идеальные Игры**: Дополнительные 500 XP
- **Ежедневные Квесты**: 200-1,000 XP каждый
- **Еженедельные Испытания**: 1,000-5,000 XP
- **Достижения**: 500-10,000 XP
- **Участие в Турнирах**: 1,000-50,000 XP

**Разблокировка Функций по Уровню**:

| Диапазон Уровней | Разблокированные Функции |
|------------|-------------------|
| 1-10 | Базовый геймплей, обучающие программы, казуальные матчи |
| 11-20 | Ранговая игра, первый слот для ИИ, ежедневные квесты |
| 21-30 | Продвинутые обучающие программы, доступ к турнирам, вступление в кланы |
| 31-40 | Доступ к студии (если имеется), возможность аренды ИИ, пользовательские комнаты |
| 41-50 | Продвинутые турниры, создание кланов, продажа на рынке |
| 51-60 | Столы с высокими ставками, премиум-функции, программа наставничества |
| 61-70 | Эксклюзивные турниры, ранний доступ к функциям |
| 71-80 | Опции престижа, доступ к легендарным предметам |
| 81-90 | Элитные соревнования, программа тестирования платформы |
| 91-100 | Максимальные преимущества, особое признание, эксклюзивный контент |

**Награды за Повышение Уровня**:
- Бонусы мягкой валюты
- Награды-талисманы
- Осколки персонажей
- Лотерейные билеты
- Косметические разблокировки
- Варианты титулов

<Callout type="success">
**Всегда Прогрессируешь**: Даже игроки максимального уровня продолжают получать награды через системы престижа и сезонные сбросы.
</Callout>

### Уровень Персонажа

Каждый персонаж накапливает опыт в матчах, открывая дополнительные функции и преимущества.

**Прогрессия Персонажа**:
- **Индивидуальный XP**: Каждый персонаж повышает уровень отдельно
- **Активный Персонаж Получает XP**: Только экипированный персонаж
- **Максимальный Уровень**: 50 для каждого персонажа
- **Опция Престижа**: Сброс до уровня 1 с постоянными бонусами

**Источники XP Персонажа**:
- Использование персонажа в матчах (основной)
- Победа с персонажем (бонус)
- Специальные испытания персонажа
- Достижения, связанные с персонажем
- Участие в событиях

**Преимущества Повышения Уровня**:
- **Косметические Опции**: Разблокируются на определенных уровнях
- **Эмоции**: Выражения, специфичные для персонажа (уровни 5, 15, 25, 35, 45)
- **Эффективность ИИ**: Улучшенная скорость обучения ИИ (+2% за уровень)
- **Бонусы к Характеристикам**: Незначительное увеличение атрибутов
- **Очки Навыков**: Распределяются по древу навыков (1 очко за уровень)

**Распределение по Древу Навыков**:

**Стратегическая Ветвь**:
- Бонусы к скорости обучения ИИ
- Разблокировка дополнительных слотов для ИИ (уровни 20, 40)
- Улучшенный анализ после игры
- Улучшение подсказок по стратегии
- Помощь в расчете вероятностей

**Социальная Ветвь**:
- Уникальные эмоции и стикеры
- Опции настройки чата
- Множители бонусов для друзей
- Увеличение вклада в клан
- Бонусы социального опыта

**Экономическая Ветвь**:
- Бонусы к заработку мягкой валюты (+1% за очко)
- Множители лотерейных билетов
- Скидки на вход в турниры (до 25%)
- Снижение комиссий на рынке
- Увеличение шансов выпадения талисманов

**Престиж Персонажа**:
- Доступен на уровне 50
- Сброс до уровня 1, но очки навыков сохраняются
- Получение постоянных бонусов к характеристикам
- Визуальные индикаторы престижа
- Эксклюзивные косметические опции
- Возможно несколько уровней престижа

### Уровень ИИ

По мере того как ИИ-компаньоны играют больше матчей, они повышают свой уровень, что приводит к улучшению производительности.

**Механики Прогрессии ИИ**:
- **Обучение на Данных**: Улучшается с большим количеством игровых данных
- **Опыт Матча**: Каждый матч способствует росту ИИ
- **Максимальный Уровень**: 50 (для каждого экземпляра ИИ)
- **Уровни Качества**: Бронза, Серебро, Золото, Платина, Бриллиант

**Преимущества Уровня ИИ**:
- **Более Быстрые Решения**: Сокращенное время размышления (на 5-50% быстрее)
- **Лучшая Оценка Риска**: Улучшенный расчет вероятностей
- **Улучшенная Адаптация**: Более быстрая корректировка под противников
- **Совершенствование Стратегии**: Более сложные игровые паттерны
- **Последовательность**: Более надежная производительность

**Требования к Обучению ИИ**:
- **Начальное Обучение**: 50-100 игр для базовой функциональности
- **Компетентность**: 200-500 игр для надежной производительности
- **Превосходство**: 1,000+ игр для ИИ высшего класса
- **Поддержка**: Непрерывная игра поддерживает ИИ в актуальном состоянии
- **Переобучение**: Обновление ИИ новыми стратегиями

**Ускорение Обучения ИИ**:
- **Тренировочные Комплексы Студии**: Более быстрое обучение
- **Учебные Модули**: Покупаемые бустеры XP
- **Симуляционные Матчи**: Учитываются в опыте ИИ
- **Целевое Обучение**: Фокусировка на конкретных слабых сторонах
- **Массовое Обучение**: Обучение нескольких ИИ одновременно (функция Студии)

<Callout type="info">
**Живой ИИ**: Ваш ИИ постоянно развивается по мере вашей игры, адаптируясь к изменениям меты и совершенствуя стратегии на основе накопленного опыта.
</Callout>

## Награды и Программы Лояльности

Для стимулирования частой игры Mahjong Stars предлагает комплексные системы вознаграждений.

### Ежедневные Квесты

Простые задачи, выполняемые в ходе обычной игры.

**Типы Квестов**:
- **Выиграть X Матчей**: Выиграть 3 матча (Награда: 1,000 мягкой валюты)
- **Сыграть Y Игр**: Завершить 5 игр независимо от исхода (Награда: 500 мягкой валюты)
- **Использовать Конкретного Персонажа**: Сыграть 3 игры с представленным персонажем (Награда: бустер XP персонажа)
- **Социальные Квесты**: Играть с друзьями, отправлять подарки (Награда: социальные предметы)
- **Квесты Варианта**: Сыграть в определенный вариант маджонга (Награда: награды, специфичные для варианта)

**Система Квестов**:
- **Ежедневный Сброс**: Новые квесты каждые 24 часа
- **Слоты Квестов**: 3-5 активных квестов одновременно
- **Опция Переброса**: Заменить один квест в день (за плату мягкой валютой)
- **Бонусы за Серию**: Последовательные дни умножают награды
- **Быстрое Завершение**: Большинство квестов выполнимы за 30-60 минут

### Еженедельные Испытания

Долгосрочные цели, требующие стратегического планирования.

**Примеры Испытаний**:
- **Выиграть с Определенными Комбинациями**: Достичь конкретных выигрышных паттернов (3 раза)
- **Достичь Вехи Ранга**: Подняться по рейтинговой лестнице (Награда: призы, основанные на ранге)
- **Результаты в Турнире**: Занять место в топ-50 еженедельного турнира
- **Цели Коллекционирования**: Приобрести 5 новых талисманов
- **Обучение ИИ**: Сыграть 50 игр с ИИ (Награда: бустер XP ИИ)

**Преимущества Еженедельных Испытаний**:
- **Более Крупные Награды**: Наборы талисманов, осколки персонажей, значительное количество мягкой валюты
- **Стратегическая Глубина**: Требует планирования и преданности
- **Фокус на Прогрессе**: Поощряет определенные стили игры
- **Общественные События**: Иногда связаны с глобальными целями

### Уровни Лояльности

Игроки накапливают очки лояльности на основе последовательных дней игры и общего количества матчей.

**Система Уровней Лояльности**:

| Уровень | Требования | Преимущества |
|------|-------------|----------|
| **Бронза** | 7 последовательных дней, 50 общих матчей | +5% мягкой валюты, 1 еженедельный лотерейный билет |
| **Серебро** | 30 последовательных дней, 200 общих матчей | +10% мягкой валюты, 2 еженедельных лотерейных билета, скидки на турниры (10%) |
| **Золото** | 90 последовательных дней, 500 общих матчей | +15% мягкой валюты, 3 еженедельных лотерейных билета, скидки на турниры (15%), эксклюзивные талисманы |
| **Платина** | 180 последовательных дней, 1,000 общих матчей | +20% мягкой валюты, 5 еженедельных лотерейных билетов, скидки на турниры (20%), приоритетная поддержка |
| **Бриллиант** | 365 последовательных дней, 2,500 общих матчей | +25% мягкой валюты, 10 еженедельных лотерейных билетов, скидки на турниры (25%), эксклюзивный доступ |

**Преимущества Лояльности**:
- **Множители Мягкой Валюты**: Постоянное увеличение заработка
- **Сниженные Сборы**: Скидки на вход в турниры
- **Доступ к Лотереям**: Автоматическое участие в эксклюзивных лотереях
- **Приоритетные Функции**: Ранний доступ к новому контенту
- **Косметические Награды**: Предметы, эксклюзивные для программы лояльности
- **Признание Статуса**: Специальные значки и титулы

**Поддержание Лояльности**:
- Достаточно ежедневного входа (матчи не требуются)
- Льготные периоды для пропущенных дней (буфер 2 дня на более высоких уровнях)
- Режим отпуска (приостановка лояльности, доступен на Золотом+ уровне)
- Бонусы за возвращение при потере серии

<Callout type="warning">
**Защита Серии**: Более высокие уровни лояльности предлагают защиту от пропущенных дней, но поддержание последовательности является ключом к максимальным преимуществам.
</Callout>

### Боевые Пропуска Событий

Сезонные события вводят боевые пропуски с прогрессивными наградами.

**Структура Боевого Пропуска**:
- **Бесплатный Уровень**: Доступен всем игрокам, скромные награды
- **Премиум Уровень**: Опциональная покупка, улучшенные награды
- **Продолжительность**: Обычно 4-8 недель на сезон
- **Уровни**: 50-100 уровней на пропуск

**Получение XP События**:
- Игра в матчи (любой режим)
- Выполнение квестов, специфичных для события
- Участие в тематических турнирах
- Бонусы за ежедневный вход
- Специальные испытания события

**Награды Бесплатного Уровня** (Примеры):
- Наборы мягкой валюты
- Обычные талисманы
- Бустеры XP персонажа
- Косметические предметы
- Лотерейные билеты

**Награды Премиум Уровня** (Дополнительно):
- Эксклюзивные персонажи
- Редкие талисманы
- Уникальная косметика
- Множители мягкой валюты
- Предметы на тему события
- Ранние награды за уровни (мгновенное получение бесплатного уровня)

**Преимущества Боевого Пропуска**:
- Сезонный контент и темы
- Эксклюзивные предложения ограниченного времени
- Общественные цели
- Дополнительные испытания
- Коллекции косметики

## Соревнования и Рейтинги Между Игроками

Соревнования выходят за рамки индивидуальной игры, способствуя развитию сообщества и соперничества.

### Рейтинги Клана

Кланы накапливают очки на основе выступлений участников в ранговых матчах и турнирах.

**Система Очков Клана**:
- **Индивидуальные Победы в Ранговых Матчах**: 10-50 очков (в зависимости от сложности противника)
- **Результаты в Турнирах**: 100-5,000 очков (в зависимости от занятого места)
- **Войны Клана**: Бонусные очки за победы один на один
- **Бонусы Активности**: Очки за участие членов клана
- **Специальные События**: Бонусные очки во время соревнований кланов

**Таблицы Лидеров Клана**:
- **Глобальные Рейтинги**: Все кланы мира
- **Региональные Рейтинги**: Соревнования по регионам
- **Сезонные Сбросы**: Новый старт каждый сезон
- **Исторические Записи**: Отслеживание наследия клана

**Награды за Рейтинг Клана**:
- **Топ-10 Кланов**: Значительное количество мягкой валюты для всех участников, эксклюзивная косметика клана
- **Топ-100 Кланов**: Умеренные награды, бонусы престижа клана
- **Топ-1000 Кланов**: Небольшие награды, значки признания
- **Участие**: Все активные кланы получают базовые награды

**Преимущества Клана**:
- Значки престижа и баннеры клана
- Бонусы мягкой валюты для участников
- Опции кастомизации
- Приоритетный доступ к турнирам
- Эксклюзивные события клана

### Межрегиональные Турниры

Специальные события, где регионы соревнуются друг с другом в дружеской конкуренции.

**Формат Региональных Соревнований**:
- **Региональные Команды**: Игроки представляют свой регион
- **Квалификация**: Отбираются лучшие игроки из каждого региона
- **Командное Начисление Очков**: Совокупный результат определяет победителя
- **Призы**: Региональная гордость + награды для всех участников

**Участвующие Регионы**:
- Восточная Азия (Китай, Япония, Корея)
- Юго-Восточная Азия (Сингапур, Таиланд, Вьетнам, Филиппины)
- Северная Америка (США, Канада)
- Европа (объединенная или по странам для крупных рынков)
- Океания (Австралия, Новая Зеландия)
- Другие (развивающиеся регионы)

**Преимущества Региональных Турниров**:
- Культурный обмен
- Региональная гордость и самобытность
- Массивные призовые фонды
- Освещение в трансляциях
- Обнаружение талантов

<Callout type="success">
**Глобальное Единство**: Межрегиональные турниры празднуют маджонг как глобальную игру, одновременно развивая здоровое региональное соперничество.
</Callout>

### Рейтинги ИИ

ИИ-компаньоны имеют свою систему рейтингов, отдельную от игроков-людей.

**Метрики Производительности ИИ**:
- **Процент Побед**: Общий процент успеха
- **Среднее Количество Очков**: Очки, заработанные за матч
- **Управление Риском**: Баланс агрессии и осторожности
- **Эффективность**: Скорость принятия решений
- **Адаптивность**: Производительность против различных противников

**Категории Рейтингов ИИ**:
- **Общие Рейтинги ИИ**: Лучшие ИИ на платформе
- **Специфичные для Варианта**: Лучшие ИИ для каждого варианта маджонга
- **Рейтинги Владельцев**: Игроки с наиболее успешными ИИ
- **Популярность Аренды**: Наиболее арендованные/востребованные ИИ
- **Темп Улучшения**: Наиболее быстро обучающиеся ИИ

**Преимущества Соревнований ИИ**:
- Стимулирует инвестиции в обучение ИИ
- Совершенствует стратегии ИИ
- Создает спрос на аренду ИИ
- Демонстрирует успех обучения
- Предоставляет эталоны для обучения

**Турниры ИИ**:
- Соревнования только для ИИ
- Демонстрация возможностей обученных ИИ
- Призы для владельцев выигрышных ИИ
- Инсайты для исследований и разработок
- Развлекательная ценность

## Философия Прогрессии

### Сбалансированное Развитие

**Принципы Дизайна**:
- **Всегда Есть Что Получить**: Множество путей прогрессии
- **Нет Тупиков**: Системы престижа и сезонные системы предотвращают ограничение
- **Навык Решает Больше Всего**: Прогресс ускоряется благодаря навыку, а не только времени
- **Доступно Всем**: Бесплатные игроки могут полностью прогрессировать
- **Опциональное Ускорение**: Премиум-опции доступны, но не обязательны

### Долгосрочное Вовлечение

**Стратегии Удержания**:
- **Ежедневные Причины Играть**: Квесты, бонусы, социальная активность
- **Еженедельные Цели**: Испытания и турниры
- **Ежемесячные Вехи**: Уровни лояльности и сезонный прогресс
- **Ежегодное Достижение**: Признание долгосрочной лояльности
- **Вечные Цели**: Завершение коллекций, системы престижа

<Callout type="info">
**Бесконечное Путешествие**: Системы прогрессии Mahjong Stars гарантируют, что всегда будет следующая цель, независимо от того, играете ли вы казуально или соревновательно.
</Callout>
