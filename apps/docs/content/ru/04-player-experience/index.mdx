# Доступность и управление учетной записью

Mahjong Stars разработан с учетом интересов игроков, обеспечивая легкий доступ, интуитивно понятное управление учетной записью и гибкие возможности для всех пользователей.

---

## Браузерная платформа и совместимость с устройствами

Mahjong Stars построен как веб-приложение, что исключает необходимость загрузки или установки. Такой подход максимизирует доступность и удобство.

### Немедленный доступ

**Загрузка не требуется**:
- Мгновенный доступ из любого современного браузера
- Без одобрения магазинов приложений и обновлений
- Не потребляет дисковое пространство устройств
- Всегда используется последняя версия
- Кроссплатформенная совместимость

**Быстрый старт**:
- Новые пользователи могут зарегистрироваться и начать играть за считанные минуты
- Онбординг без трения
- Без ожидания крупных загрузок
- Немедленный доступ к игровому процессу

<Callout type="success">
**Платформенно-независимая**: Работает на Windows, macOS, Linux, iOS, Android и любом устройстве с современным веб-браузером.
</Callout>

### Кроссплатформенная игра

Адаптивный дизайн гарантирует, что интерфейс подстраивается под различные размеры экрана и устройства.

**Бесшовный опыт**:
- **Настольный компьютер/ноутбук**: Полнофункциональный опыт с оптимальной компоновкой
- **Планшет**: Интерфейс, оптимизированный для сенсорного ввода с поддержкой жестов
- **Мобильные устройства**: Оптимизированный пользовательский интерфейс для небольших экранов
- **Непрерывная игра**: Начните на одном устройстве, продолжите на другом
- **Облачная синхронизация**: Прогресс автоматически сохраняется на всех устройствах

**Оптимизация для конкретных устройств**:
- **Сенсорное управление**: Естественные жесты для мобильных устройств и планшетов
- **Мышь и клавиатура**: Точное управление на настольных компьютерах
- **Поддержка геймпадов**: Опциональная интеграция контроллера
- **Поворот экрана**: Режимы ландшафтной и портретной ориентации
- **Интеллектуальное масштабирование**: Адаптивные элементы пользовательского интерфейса для всех размеров экрана

### Низкие аппаратные требования

Поскольку интенсивные вычисления переносятся на облачные серверы, Mahjong Stars плавно работает на устройствах со скромными характеристиками.

**Минимальные требования**:
- **Браузер**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **ОЗУ**: Минимум 2 ГБ, рекомендуется 4 ГБ
- **Интернет**: Минимум 2 Мбит/с, рекомендуется 5 Мбит/с
- **ЦП**: Любой современный двухъядерный процессор
- **Графика**: Базовая видеокарта или интегрированная графика

**Преимущества производительности**:
- Старые устройства могут запускать платформу
- Игровое оборудование не требуется
- Сниженное энергопотребление
- Более низкий порог входа
- Более широкая доступность на рынке

<Callout type="info">
**Облачная мощь**: Серверная обработка означает, что ваше устройство обрабатывает только отображение и ввод, делая платформу доступной практически для любого современного устройства.
</Callout>

## Создание учетной записи и интеграция кошелька

Создание учетной записи в Mahjong Stars — это простой процесс с несколькими вариантами, отвечающими различным предпочтениям пользователей.

### Стандартные методы регистрации

**Электронная почта и пароль**:
- Традиционная регистрация с подтверждением по электронной почте
- Требования к надежному паролю
- Варианты восстановления пароля
- Настройки уведомлений по электронной почте

**Вход через социальные сети**:
- **Google**: Быстрая интеграция OAuth
- **Facebook**: Подключение существующей учетной записи
- **Apple**: Вход с акцентом на конфиденциальность
- **Twitter/X**: Интеграция социальных игр
- **Discord**: Подход, ориентированный на сообщество

**Преимущества входа через социальные сети**:
- Более быстрый процесс регистрации
- Не нужно запоминать пароль
- Упрощенное восстановление учетной записи
- Интеграция с социальной сетью
- Доверие через установленные платформы

### Настройка кошелька (необязательно)

Хотя учетные записи с мягкой валютой создаются автоматически, игроки, желающие торговать цифровыми активами или взаимодействовать с экосистемой токенов, могут подключить совместимый кошелек.

<Callout type="warning">
**Полностью необязательно**: Интеграция с кошельком НЕ требуется для игры в Mahjong Stars. Она нужна только для функций блокчейна, таких как торговля NFT и транзакции с токенами.
</Callout>

**Поддерживаемые кошельки**:
- **MetaMask**: Самый популярный кошелек Ethereum
- **Trust Wallet**: Удобный мобильный мультичейновый кошелек
- **Coinbase Wallet**: Вариант для начинающих
- **WalletConnect**: Универсальное подключение кошелька
- **Нативный веб-кошелек**: Встроенный вариант для простоты

**Процесс настройки кошелька**:
1. Создайте учетную запись по электронной почте или через социальные сети
2. Перейдите в настройки кошелька (необязательно)
3. Выберите поставщика кошелька
4. Подключите существующий кошелек или создайте новый
5. Подтвердите подключение
6. Начните использовать функции блокчейна

**Руководства на платформе**:
- Пошаговые руководства по кошелькам
- Видеоуроки
- Лучшие практики безопасности
- Рекомендации по аппаратным кошелькам
- Информация о важности сид-фразы

### Функции безопасности

Безопасность учетной записи имеет первостепенное значение в Mahjong Stars.

**Меры безопасности**:
- **Двухфакторная аутентификация (2FA)**: SMS или приложение-аутентификатор
- **Подтверждение по электронной почте**: Подтвердите адрес электронной почты для действий с учетной записью
- **Оповещения о входе**: Уведомления о входе с новых устройств
- **Управление сеансами**: Просмотр и отзыв активных сеансов
- **Биометрический вход**: Отпечаток пальца/Face ID на мобильных устройствах
- **Журналы безопасности**: Отслеживание активности учетной записи
- **Требования к паролю**: Принудительное применение надежных паролей

**Рекомендуемые лучшие практики**:
- Используйте уникальные пароли
- Сразу включите 2FA
- Не делитесь учетными данными
- Используйте аппаратные кошельки для крупных вложений
- Регулярное обновление паролей
- Безопасное хранение сид-фразы

## Варианты валюты

Mahjong Stars использует двойную валютную систему, предназначенную для всех типов игроков.

### Мягкая валюта (игровые кредиты)

Игровые кредиты, поддерживающие повседневный игровой процесс, доступные всем игрокам без криптовалютных требований.

<Callout type="success">
**Бесплатный заработок**: Мягкая валюта в основном зарабатывается в процессе игры, что делает Mahjong Stars действительно бесплатной игрой.
</Callout>

**Заработок мягкой валюты**:
- **Победы в матчах**: Основной способ заработка
- **Ежедневные квесты**: Выполняйте задания для получения наград
- **Еженедельные испытания**: Долгосрочные цели
- **Достижения**: Награды за этапы
- **Турниры**: Призы за соревнования
- **Бонусы за вход**: Ежедневные награды для активных игроков
- **Заработок ИИ**: Ваш ИИ зарабатывает, пока вы не в сети
- **Бонусы очарования**: Повышение скорости заработка

**Использование мягкой валюты**:
- Покупка персонажей в основном магазине
- Покупка очарований и косметики
- Вход в турниры (вступительные взносы)
- Ускорение обучения ИИ
- Настройка студий
- Покупка тематических наборов
- Покупка билетов на розыгрыш
- Кастомизация столов

**Свойства мягкой валюты**:
- Не может быть напрямую выведена в фиат
- Неспекулятивный дизайн
- Фиксированный обменный курс для покупок
- Исключает механику "плати, чтобы выиграть"
- Защита от инфляции

### Жесткая валюта (токен управления)

Токен на основе блокчейна для игроков, которые хотят участвовать в управлении платформой и функциях торговой площадки.

**Сценарии использования токена**:
- **Голосование по управлению**: Участие в решениях платформы
- **Транзакции на торговой площадке**: Торговля активами между пользователями
- **Стейкинг**: Получение наград и разблокировка преимуществ
- **Премиум-функции**: Доступ к эксклюзивному контенту
- **Торговая площадка ИИ**: Покупка/продажа обученных моделей ИИ
- **Вступительные взносы в турниры**: Соревнования с высокими ставками
- **Покупка активов**: Особые коллекционные предметы

**Получение токенов**:
- Покупка на биржах
- Заработок в турнирах высокого уровня
- Награды за стейкинг
- Награды за участие в управлении
- Прибыль от торговли на торговой площадке
- Специальные мероприятия и аирдропы

<Callout type="info">
**Отдельная сущность**: Токен управления выпускается и управляется отдельной организацией, что позволяет Mahjong Stars работать независимо от крипто-регулирования.
</Callout>

### Депозиты и покупки

**Покупка мягкой валюты**:
- **Фиатные платежи**: Кредитные/дебетовые карты, PayPal
- **Конвертация токенов**: Обмен токенов на мягкую валюту
- **Пакетные предложения**: Различные номиналы в наличии
- **Бонусные предложения**: Бонусы для первых покупателей
- **Безопасные платежи**: Обработка платежей, соответствующая PCI

**Варианты оплаты**:
- Кредитные карты (Visa, Mastercard, Amex)
- Дебетовые карты
- PayPal
- Apple Pay / Google Pay
- Криптовалюта (через конвертацию токенов)
- Региональные способы оплаты (Alipay, WeChat Pay и т. д.)

**Безопасность**:
- Шифрованная обработка платежей
- Системы обнаружения мошенничества
- Политики защиты покупателей
- Процедуры возврата средств
- История безопасных транзакций

### Погашение и вывод средств

**Погашение мягкой валюты**:
Мягкая валюта предназначена в первую очередь для использования в игре и не может быть выведена напрямую в фиате. Однако игроки имеют косвенные варианты погашения:

**Путь погашения**:
1. Используйте мягкую валюту для покупки цифровых активов (персонажи, очарования)
2. Конвертируйте активы в NFT (если включен блокчейн)
3. Продавайте активы на торговой площадке за токены или фиат
4. Используйте сторонние биржи для окончательной конвертации

**Эта модель**:
- Предотвращает возможное злоупотребление
- Поддерживает здоровую экономику
- Предоставляет путь извлечения стоимости
- Требует выбора и усилий игрока
- Соответствует нормативным требованиям

<Callout type="warning">
**Не система вывода денег**: Mahjong Stars — это игровая платформа, а не финансовый инструмент. Погашение возможно через стоимость активов, но не является основной мотивацией.
</Callout>

### Конвертация валюты

**Мягкая валюта ↔ Токен**:
- Конвертируйте токены в мягкую валюту по рыночным курсам
- Односторонняя конвертация (только токены → мягкая валюта)
- Предотвращает дефляцию токенов
- Поддерживает стабильность мягкой валюты
- Отображаются четкие обменные курсы

**Определение курса**:
- Рыночное ценообразование
- Прозрачный расчет
- Без скрытых комиссий
- Обновления в реальном времени
- Отслеживание исторических курсов

## Функции управления учетной записью

### Управление профилем

**Настраиваемый профиль**:
- Имя пользователя и отображаемое имя
- Выбор аватара (из принадлежащих персонажей)
- Биография и описание
- Ссылки на социальные сети
- Настройки конфиденциальности
- Варианты отображения статистики

### Настройки и предпочтения

**Настройки игры**:
- Вариант игры по умолчанию
- Частота подсказок ИИ
- Предпочтения автоматического подбора
- Настройки кастомизации стола по умолчанию
- Предпочтения уведомлений

**Настройки конфиденциальности**:
- Видимость профиля (публичный/друзья/приватный)
- Видимость истории матчей
- Разрешения на запросы в друзья
- Доступность чата
- Отображение статуса онлайн
- Предпочтения сбора данных (соответствие GDPR)

**Предпочтения связи**:
- Уведомления по электронной почте
- Push-уведомления
- Оповещения в игре
- Маркетинговые сообщения
- Обновления активности друзей
- Напоминания о турнирах

### Родительский контроль

Для юных игроков или семейных учетных записей:

**Функции контроля**:
- Лимиты расходов
- Ограничения времени игры
- Ограничения чата
- Лимиты запросов в друзья
- Фильтры контента
- Отчеты об активности для опекунов
