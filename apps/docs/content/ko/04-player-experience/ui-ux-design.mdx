# UI/UX 디자인 & 플레이어 참여

깔끔하고 직관적인 인터페이스는 새로운 플레이어와 숙련된 플레이어 모두에게 마작 스타즈를 환영받는 게임으로 만듭니다. 디자인은 몰입감 있는 게임 경험을 만드는 동시에 명확성, 최소한의 산만함, 접근 가능한 메뉴에 중점을 둡니다.

---

## 직관적인 인터페이스 및 다국어 지원

### 대시보드 디자인

홈 화면은 플레이어가 한눈에 필요한 모든 것을 요약해 줍니다.

**대시보드 요소**:
- **플레이어 카드**: 아바타, 레벨, 랭크 및 빠른 통계
- **빠른 플레이 버튼**: 즉각적인 매치메이킹 액세스
- **온라인 친구**: 누가 접속해 있는지 확인
- **활성 이벤트**: 현재 진행 중인 토너먼트 및 프로모션
- **최근 경기**: 빠른 다시 보기 액세스
- **알림**: 업데이트 및 메시지
- **일일 퀘스트**: 진행 상황 추적
- **추천 콘텐츠**: 새로운 캐릭터, 부적, 업데이트

**레이아웃 원칙**:
- **깔끔한 디자인**: 시각적인 산만함 없음
- **정보 계층 구조**: 중요한 항목이 눈에 띄게 표시
- **빠른 액세스**: 일반적인 작업에 쉽게 도달
- **반응형**: 화면 크기에 맞게 조정
- **사용자 정의 가능**: 플레이어가 요소 배열 가능

<Callout type="success">
**첫인상**: 대시보드는 환영하고 정리된 첫인상을 만들어 플레이어를 플랫폼을 통해 직관적으로 안내합니다.
</Callout>

### 게임 로비

매치 생성 및 검색이 효율성을 위해 간소화되었습니다.

**로비 기능**:
- **빠른 매치**: 원클릭 매치메이킹
- **사용자 정의 필터**: 특정 게임 유형 검색
- **방 찾아보기**: 사용 가능한 테이블 확인
- **방 만들기**: 간단한 방 생성 마법사
- **토너먼트 목록**: 진행 중인 대회
- **친구 초대**: 쉬운 소셜 플레이 설정

**필터링 옵션**:
- **변형**: 리치, 방방 등
- **판돈**: 소량 통화 범위
- **난이도**: 초급부터 전문가까지
- **게임 속도**: 빠름, 보통 또는 느긋함
- **비공개/공개**: 방 가시성
- **지역**: 지리적 선호도

**검색 및 정렬**:
- 방 이름 또는 호스트로 검색
- 플레이어 수, 판돈, 생성 시간별 정렬
- 특정 설정 즐겨찾기
- 최근 방 기록
- 실력 기반 추천 매치

### 인벤토리 및 컬렉션

전용 인벤토리 페이지에 소유한 모든 자산이 표시됩니다.

**정리 기능**:
- **카테고리 탭**: 캐릭터, 부적, 스튜디오
- **정렬 옵션**: 희귀도, 레벨, 획득 날짜, 이름
- **필터 기능**: 자산 빠른 검색
- **그리드/목록 보기**: 표시 선호도
- **비교 도구**: 자산 측면 비교
- **빠른 작업**: 장착, 업그레이드, 판매 버튼

**자산 관리**:
- 작업을 위한 대량 선택
- 일괄 판매/전환
- 즐겨찾기 표시
- 사용자 정의 정리 폴더
- 속성별 검색
- 컬렉션 진행 상황 추적

### 설정 및 사용자 정의

개인화된 경험을 위한 포괄적인 설정입니다.

**설정 카테고리**:

**시각 설정**:
- 그래픽 품질 (낮음~울트라)
- 해상도 및 전체 화면 옵션
- 프레임 속도 제한
- 애니메이션 속도
- 파티클 효과
- 테마 선택

**오디오 설정**:
- 마스터 볼륨
- 음악, SFX, 음성 개별 제어
- 오디오 출력 장치 선택
- 서라운드 사운드 옵션
- 배경 음악 재생 목록

**게임 플레이 설정**:
- AI 힌트 빈도
- 자동 매치 설정
- 카메라 앵글
- 패 모양 강조
- 취소 권한 (캐주얼만 해당)
- 자동 타일 정렬

**채팅 환경 설정**:
- 채팅 가시성
- 비속어 필터 수준
- 채팅 글꼴 크기
- 알림 소리
- 차단된 사용자 목록
- 빠른 채팅 사용자 정의

<Callout type="info">
**접근성 우선**: 설정에는 색맹 모드, 큰 글꼴, 모바일 햅틱 피드백 및 기타 여러 접근성 기능이 포함됩니다.
</Callout>

### 다국어 지원

전 세계 사용자를 수용하기 위해 인터페이스는 여러 언어로 제공됩니다.

**출시 언어**:
- 영어 (미국 및 영국)
- 중국어 간체 (만다린)
- 중국어 번체
- 일본어
- 한국어
- 스페인어
- 포르투갈어 (브라질)
- 프랑스어
- 독일어
- 러시아어
- 아랍어

**추가 언어** (단계적 출시):
- 이탈리아어, 네덜란드어, 폴란드어
- 태국어, 베트남어, 인도네시아어
- 터키어, 힌디어
- 사용자 수요에 따른 추가 언어

**현지화 기능**:
- **전체 번역**: UI, 튜토리얼, 도움말 문서
- **문화 적응**: 지역에 적합한 콘텐츠
- **숫자/날짜 형식**: 현지화된 형식
- **결제 옵션**: 지역별 결제 수단
- **지원**: 원어 고객 서비스
- **커뮤니티**: 언어별 채널

**언어 설정**:
- 쉬운 언어 전환
- 위치 기반 자동 감지
- 별도의 채팅 언어 환경 설정
- 글로벌 채팅을 위한 번역 도구
- 언어 학습 자료

## 대화형 튜토리얼 및 초보자 친화적 기능

마작 스타즈는 마작의 복잡성이 새로운 플레이어를 단념시킬 수 있다는 것을 인지하고 있습니다. 포괄적인 교육이 플랫폼에 내장되어 있습니다.

### 튜토리얼 모듈

단계별 튜토리얼은 기본 규칙, 패 구성, 채점 및 타일 흐름을 소개합니다.

**튜토리얼 구조**:

**레벨 1 - 기초**:
- 타일 및 문양 소개
- 멜드 이해 (퐁, 치, 깡)
- 패 구성 요구 사항
- 승리 조건
- 기본 채점

**레벨 2 - 전략**:
- 테이블 읽기
- 방어 플레이 개념
- 멜드를 선언할 때
- 확률 기초
- 위험 관리

**레벨 3 - 변형**:
- 리치 규칙 및 전략
- 방방 게임 플레이
- 변형별 메커니즘
- 특수 패
- 고급 채점

**레벨 4 - 플랫폼 기능**:
- AI 힌트 사용
- 토너먼트 참여
- 캐릭터 및 부적 시스템
- 소셜 기능
- 마켓플레이스 기초

**튜토리얼 기능**:
- **대화형**: 단순히 읽는 것이 아니라 직접 연습
- **반복 가능**: 언제든지 다시 볼 수 있음
- **보상**: 완료 시 소량 통화 제공
- **진행 상황 추적**: 학습 여정 확인
- **게임화**: 튜토리얼 완료 시 업적

<Callout type="success">
**초보에서 고수로**: 튜토리얼 시스템은 완전한 초보자를 몇 시간의 집중적인 학습을 통해 능숙한 플레이어로 만들 수 있습니다.
</Callout>

### 안내된 첫 경기

새로운 플레이어는 AI 힌트가 영구적으로 활성화된 상태로 첫 몇 경기를 플레이할 수 있습니다.

**첫 경기 경험**:
- **필수 튜토리얼 경기**: 건너뛸 수 없으며 부드러운 소개
- **안내 경기 (5-10판)**: 힌트 및 설명 강화
- **초보자 보호**: 다른 초보자와만 매치
- **관대한 규칙**: 더 긴 턴 타이머, 취소 옵션
- **격려**: 긍정적인 강화 메시지

**힌트 설명**:
- 무엇을 해야 하는지뿐만 아니라 왜 해야 하는지도 설명
- 확률 설명
- 전략적 이유
- 학습 목표 강조
- 실력 향상에 따른 점진적인 힌트 감소

### 용어집 및 지식 기반

인앱 용어집은 마작 용어, 일반적인 전략 및 속어를 정의합니다.

**지식 기반 기능**:
- **검색 가능한 데이터베이스**: 모든 용어를 빠르게 찾기
- **하이퍼링크 콘텐츠**: 관련 용어 연결
- **시각적 예시**: 이미지 및 다이어그램
- **비디오 튜토리얼**: 복잡한 개념 시각적으로 설명
- **전략 가이드**: 전문가 작성
- **커뮤니티 기여**: 플레이어가 제출한 팁

**콘텐츠 카테고리**:
- 마작 용어
- 플랫폼 기능
- 전략 가이드
- 캐릭터 가이드
- AI 훈련 팁
- 토너먼트 전략
- 문화적 맥락

**통합**:
- 게임 내 툴팁
- 상황별 도움말
- 빠른 참조 팝업
- 튜토리얼에서 링크
- 지식 기사 공유

### 초보자 로비

전용 초보자 방은 신규 플레이어가 비슷한 실력의 다른 플레이어와 경쟁하도록 보장합니다.

**초보자 구역 기능**:
- **레벨 제한**: 신규 플레이어만 이용 가능 (첫 50경기)
- **지원 환경**: 친근한 분위기 장려
- **판돈 없음**: 무료 플레이, 부담 없이 배우기
- **인내심 있는 속도**: 더 긴 턴 타이머
- **도우미 플레이어**: 자원봉사 멘토 이용 가능
- **졸업 시스템**: 준비되면 메인 로비로 "레벨 업"

## 몰입감 있는 게임 플레이 및 시각적 요소

시각 및 청각 디자인은 마작 스타즈 경험에 크게 기여합니다.

### 테마 선택

플레이어는 전통적인 테마와 현대적인 디자인 중에서 선택할 수 있습니다.

**사용 가능한 테마**:

**전통 테마**:
- **대나무 숲**: 고요한 자연 환경
- **벚꽃**: 일본 미학
- **황궁**: 중국 고전
- **찻집**: 아늑한 응접실 분위기
- **선(禪) 정원**: 평화로운 미니멀리즘

**현대 테마**:
- **사이버펑크 도시**: 네온 미래
- **우주 정거장**: SF 환경
- **추상 미술**: 현대 디자인
- **미니멀리즘**: 깔끔하고 단순한 미학
- **다크 모드**: 눈의 피로 감소

**시즌 테마** (한정 시간):
- 설날
- 여름 해변
- 가을 수확
- 겨울 원더랜드
- 명절 스페셜

<Callout type="info">
**혼합 및 일치**: 플레이어는 다른 테마의 요소를 결합하여 개인화된 미학을 만들 수 있습니다.
</Callout>

### 동적 애니메이션

플레이어가 멜드를 선언하거나 승리할 때 애니메이션은 게임 플레이를 방해하지 않고 흥미를 더합니다.

**애니메이션 유형**:
- **타일 이동**: 부드럽고 사실적인 물리 효과
- **멜드 선언**: 만족스러운 시각적 피드백
- **승리 애니메이션**: 축하 효과
- **캐릭터 반응**: 아바타가 게임 플레이에 반응
- **특수 패**: 희귀 승리를 위한 고유 애니메이션

**애니메이션 제어**:
- 속도 조절
- 특정 유형 비활성화
- 움직임 감소 모드
- 애니메이션 건너뛰기 옵션
- 품질 설정

### 사운드 디자인

배경 음악과 음향 효과는 테마를 보완하고 몰입감을 향상시킵니다.

**오디오 요소**:

**음악**:
- **앰비언트 트랙**: 테마 선택과 일치
- **동적 음악**: 게임 상태에 따라 강도 조절
- **사용자 정의 재생 목록**: 좋아하는 트랙 선택
- **고전 아시아 악기**: 전통적인 느낌
- **현대 전자 음악**: 현대적인 옵션
- **볼륨 믹서**: 개별 트랙 제어

**사운드 효과**:
- **타일 소리**: 만족스러운 클릭 및 찰칵거림
- **멜드 선언**: 오디오 피드백
- **턴 표시기**: 미묘한 경고음
- **승리 소리**: 축하 오디오
- **캐릭터 음성**: 개성 및 풍미

**성우 연기**:
- 캐릭터별 대사
- 다양한 언어 옵션
- 전문 성우
- 상황별 응답
- 개성 표현

**오디오 사용자 정의**:
- 사용자 정의 사운드 팩 선택
- 개인 사운드 업로드 (가이드라인 준수)
- 커뮤니티 사운드 팩
- 특정 요소 음소거
- 다양한 분위기에 맞는 오디오 프로필

### 카메라 앵글

조정 가능한 시점을 통해 플레이어는 자신의 패에 집중하거나 전체 테이블을 볼 수 있습니다.

**보기 옵션**:
- **1인칭**: 자신의 패에 집중
- **조감도**: 전체 테이블 보기
- **3인칭**: 캐릭터 중심 시점
- **자유 카메라**: 수동 제어 (다시 보기 모드)
- **시네마틱**: 자동 조정되는 극적인 각도

**카메라 기능**:
- 부드러운 전환
- 빠른 전환 단축키
- 선호하는 각도 저장
- 변형별 환경 설정
- 줌 컨트롤

**시네마틱 다시 보기**:
- 다중 각도 하이라이트
- 슬로우 모션 중요 순간
- 극적인 승리 패
- 자동 생성된 최고의 플레이
- 공유 최적화 클립

### 디자인 철학

**핵심 UX 원칙**:

2. **효율성**: 모든 기능에 대한 빠른 액세스
3. **일관성**: 전체적으로 익숙한 패턴
4. **피드백**: 행동에 대한 즉각적인 반응
5. **용서**: 쉬운 오류 복구

**속도가 중요**:
- 빠른 페이지 로드 (< 2초)

- 부드러운 애니메이션 (60 FPS 목표)
- 최적화된 자산
- 점진적 로딩
- 캐싱 전략

- 분석 기반 의사 결정

- 반복적인 디자인 업데이트
