# 접근성 및 계정 관리

마작 스타즈는 모든 플레이어가 쉽게 액세스하고 직관적으로 계정을 관리하며 유연한 옵션을 사용할 수 있도록 플레이어를 중심으로 설계되었습니다.

---

## 브라우저 기반 플랫폼 및 기기 호환성

마작 스타즈는 웹 애플리케이션으로 구축되어 다운로드나 설치가 필요 없습니다. 이 접근 방식은 접근성과 편의성을 극대화합니다.

### 즉시 액세스

**다운로드 불필요**:
- 모든 최신 브라우저에서 즉시 액세스
- 앱 스토어 승인 또는 업데이트 불필요
- 기기 저장 공간 소비 불필요
- 항상 최신 버전으로 실행
- 크로스 플랫폼 호환성

**빠른 시작**:
- 신규 사용자는 몇 분 안에 등록하고 플레이할 수 있습니다.
- 제로 마찰 온보딩
- 대용량 다운로드 대기 시간 없음
- 즉시 게임 플레이 액세스

<Callout type="success">
**플랫폼 독립적**: Windows, macOS, Linux, iOS, Android 및 최신 웹 브라우저가 있는 모든 기기에서 작동합니다.
</Callout>

### 크로스 디바이스 플레이

반응형 디자인은 인터페이스가 다양한 화면 크기와 기기에 적응하도록 보장합니다.

**원활한 경험**:
- **데스크탑/랩톱**: 최적화된 레이아웃으로 모든 기능을 갖춘 경험
- **태블릿**: 제스처 지원이 포함된 터치 최적화 인터페이스
- **모바일**: 더 작은 화면에 최적화된 UI
- **연속 플레이**: 한 기기에서 시작하여 다른 기기에서 계속 플레이
- **클라우드 동기화**: 모든 기기에서 진행 상황이 자동으로 저장됩니다.

**기기별 최적화**:
- **터치 컨트롤**: 모바일 및 태블릿을 위한 자연스러운 제스처
- **마우스 및 키보드**: 데스크탑에서 정밀한 제어
- **게임패드 지원**: 선택적 컨트롤러 통합
- **화면 회전**: 가로 모드 및 세로 모드
- **스마트 스케일링**: 모든 화면 크기에 대한 적응형 UI 요소

### 낮은 하드웨어 요구 사항

무거운 연산은 클라우드 서버로 오프로드되므로 마작 스타즈는 사양이 낮은 기기에서도 원활하게 실행됩니다.

**최소 요구 사항**:
- **브라우저**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **RAM**: 최소 2GB, 권장 4GB
- **인터넷**: 최소 2Mbps, 권장 5Mbps
- **CPU**: 모든 최신 듀얼 코어 프로세서
- **그래픽**: 기본 GPU 또는 통합 그래픽

**성능 이점**:
- 이전 기기에서도 플랫폼 실행 가능
- 게임용 하드웨어 불필요
- 전력 소비 감소
- 진입 장벽 감소
- 더 넓은 시장 접근성

<Callout type="info">
**클라우드 기반**: 서버 측 처리를 통해 기기는 디스플레이와 입력만 처리하므로 거의 모든 최신 기기에서 플랫폼에 액세스할 수 있습니다.
</Callout>

## 계정 생성 및 지갑 통합

마작 스타즈 계정 생성은 다양한 사용자 선호도에 맞춘 여러 옵션을 통해 간단합니다.

### 표준 가입 방법

**이메일 및 비밀번호**:
- 이메일 인증을 통한 전통적인 등록
- 강력한 비밀번호 요구 사항
- 비밀번호 복구 옵션
- 이메일 알림 기본 설정

**소셜 미디어 로그인**:
- **Google**: 빠른 OAuth 통합
- **Facebook**: 기존 계정 연결
- **Apple**: 개인 정보 보호 중심 로그인
- **Twitter/X**: 소셜 게임 통합
- **Discord**: 커뮤니티 중심 접근 방식

**소셜 로그인 이점**:
- 더 빠른 등록 프로세스
- 기억해야 할 비밀번호 없음
- 더 쉬운 계정 복구
- 소셜 그래프 통합
- 확립된 플랫폼을 통한 신뢰

### 지갑 설정 (선택 사항)

소프트 통화 계정은 자동으로 생성되지만, 디지털 자산을 거래하거나 토큰 생태계와 상호 작용하려는 플레이어는 호환되는 지갑을 연결할 수 있습니다.

<Callout type="warning">
**전적으로 선택 사항**: 마작 스타즈를 플레이하는 데 지갑 통합이 필요하지 않습니다. NFT 거래 및 토큰 거래와 같은 블록체인 기능에만 필요합니다.
</Callout>

**지원되는 지갑**:
- **MetaMask**: 가장 인기 있는 이더리움 지갑
- **Trust Wallet**: 모바일 친화적인 멀티체인 지갑
- **Coinbase Wallet**: 초보자 친화적인 옵션
- **WalletConnect**: 범용 지갑 연결
- **기본 웹 지갑**: 단순화를 위한 내장 옵션

**지갑 설정 프로세스**:
1. 이메일 또는 소셜 로그인을 사용하여 계정 생성
2. 지갑 설정으로 이동 (선택 사항)
3. 지갑 공급자 선택
4. 기존 지갑 연결 또는 새 지갑 생성
5. 연결 확인
6. 블록체인 기능 사용 시작

**인앱 가이드**:
- 단계별 지갑 튜토리얼
- 비디오 가이드
- 보안 모범 사례
- 하드웨어 지갑 권장 사항
- 복구 구문 중요 교육

### 보안 기능

계정 보안은 마작 스타즈에서 가장 중요합니다.

**보안 조치**:
- **2단계 인증(2FA)**: SMS 또는 인증 앱
- **이메일 인증**: 계정 작업에 대한 이메일 확인
- **로그인 알림**: 새 기기 로그인 알림
- **세션 관리**: 활성 세션 보기 및 해지
- **생체 인식 로그인**: 모바일에서 지문/Face ID
- **보안 로그**: 계정 활동 추적
- **비밀번호 요구 사항**: 강력한 비밀번호 시행

**권장 모범 사례**:
- 고유한 비밀번호 사용
- 즉시 2FA 활성화
- 계정 정보 공유 금지
- 대량 보유 시 하드웨어 지갑 사용
- 정기적인 비밀번호 업데이트
- 복구 구문 안전하게 저장

## 통화 옵션

마작 스타즈는 모든 유형의 플레이어를 수용하도록 설계된 이중 통화 시스템을 사용합니다.

### 소프트 통화 (인게임 크레딧)

모든 플레이어가 암호화폐 요구 사항 없이 액세스할 수 있는 일상적인 게임 플레이를 지원하는 게임 내 크레딧입니다.

<Callout type="success">
**무료 획득**: 소프트 통화는 주로 게임 플레이를 통해 획득되므로 마작 스타즈는 진정한 무료 플레이 게임입니다.
</Callout>

**소프트 통화 획득**:
- **매치 승리**: 주요 획득 방법
- **일일 퀘스트**: 보상을 위한 목표 달성
- **주간 챌린지**: 장기 목표
- **업적**: 마일스톤 보상
- **토너먼트**: 경쟁 상금
- **로그인 보너스**: 활동적인 플레이어를 위한 일일 보상
- **AI 수입**: 오프라인 중 AI가 수입을 올립니다.
- **부적 보너스**: 수입 증가율

**소프트 통화 사용**:
- 기본 상점에서 캐릭터 구매
- 부적 및 꾸미기 아이템 구매
- 토너먼트 참가 (참가비)
- AI 훈련 가속화
- 스튜디오 맞춤 설정
- 테마 팩 구매
- 복권 티켓 구매
- 테이블 맞춤 설정

**소프트 통화 속성**:
- 직접 법정 화폐로 인출 불가
- 투기 방지 설계
- 구매 시 고정 환율
- 페이투윈(Pay-to-win) 메커니즘 방지
- 인플레이션 방지

### 하드 통화 (거버넌스 토큰)

플랫폼 거버넌스 및 마켓플레이스 기능에 참여하려는 플레이어를 위한 블록체인 기반 토큰입니다.

**토큰 사용 사례**:
- **거버넌스 투표**: 플랫폼 결정에 참여
- **마켓플레이스 거래**: P2P 자산 거래
- **스테이킹**: 보상 획득 및 혜택 잠금 해제
- **프리미엄 기능**: 독점 콘텐츠 액세스
- **AI 마켓플레이스**: 훈련된 AI 모델 구매/판매
- **토너먼트 참가**: 고위험 경쟁
- **자산 구매**: 특별 수집품

**토큰 획득**:
- 거래소에서 구매
- 최고 레벨 토너먼트를 통해 획득
- 스테이킹 보상
- 거버넌스 참여 보상
- 마켓플레이스 거래 수익
- 특별 이벤트 및 에어드롭

<Callout type="info">
**별도 법인**: 거버넌스 토큰은 별도 법인에서 발행 및 관리하므로 마작 스타즈는 암호화폐 규제와 독립적으로 게임을 운영할 수 있습니다.
</Callout>

### 입금 및 구매

**소프트 통화 구매**:
- **법정 화폐 결제**: 신용/직불 카드, PayPal
- **토큰 전환**: 토큰을 소프트 통화로 교환
- **패키지 상품**: 다양한 액수 제공
- **보너스 제안**: 첫 구매자 보너스
- **안전한 결제**: PCI 준수 결제 처리

**결제 옵션**:
- 신용 카드 (Visa, Mastercard, Amex)
- 직불 카드
- PayPal
- Apple Pay / Google Pay
- 암호화폐 (토큰 전환을 통해)
- 지역 결제 방법 (Alipay, WeChat Pay 등)

**보안**:
- 암호화된 결제 처리
- 사기 탐지 시스템
- 구매자 보호 정책
- 환불 절차
- 안전한 거래 내역

### 환전 및 인출

**소프트 통화 환전**:
소프트 통화는 주로 게임 내 사용을 위한 것이며 직접 법정 화폐로 인출할 수 없습니다. 그러나 플레이어는 간접적인 환전 옵션을 사용할 수 있습니다.

**환전 경로**:
1. 소프트 통화를 사용하여 디지털 자산 (캐릭터, 부적) 구매
2. 자산을 NFT로 변환 (블록체인 활성화 시)
3. 마켓플레이스에서 자산을 토큰 또는 법정 화폐로 판매
4. 제3자 거래소를 사용하여 최종 전환

**이 모델**:
- 잠재적 남용 방지
- 건강한 경제 유지
- 가치 추출 경로 제공
- 플레이어의 선택과 노력 필요
- 규정 준수

<Callout type="warning">
**현금화 시스템 아님**: 마작 스타즈는 게임 플랫폼이지 금융 상품이 아닙니다. 자산 가치를 통해 환전이 가능하지만, 주요 동기로 설계된 것은 아닙니다.
</Callout>

### 통화 전환

**소프트 통화 ↔ 토큰**:
- 시장 환율로 토큰을 소프트 통화로 전환
- 단방향 전환 (토큰 → 소프트 통화만 해당)
- 토큰 디플레이션 방지
- 소프트 통화 안정성 유지
- 명확한 환율 표시

**환율 결정**:
- 시장 기반 가격 책정
- 투명한 계산
- 숨겨진 수수료 없음
- 실시간 업데이트
- 과거 환율 추적

## 접근성 기능

### 시각적 접근성

**옵션**:
- 색맹 모드 (다양한 유형)
- 고대비 테마
- 조절 가능한 글꼴 크기
- 화면 판독기 지원
- 확장 가능한 UI 요소
- 애니메이션 축소 옵션

### 오디오 접근성

**기능**:
- 오디오 이벤트에 대한 시각적 단서
- 음성 라인에 대한 자막 옵션
- 조절 가능한 오디오 믹싱
- 모노 오디오 옵션
- 햅틱 피드백 (모바일)

### 입력 접근성

**지원**:
- 키보드 전용 탐색
