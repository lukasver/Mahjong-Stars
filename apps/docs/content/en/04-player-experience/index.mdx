# Accessibility & Account Management

Mahjong Stars is designed with players at the centre, ensuring easy access, intuitive account management, and flexible options for all users.

---

## Browser-Based Platform and Device Compatibility

Mahjong Stars is built as a web application, eliminating the need for downloads or installations. This approach maximizes accessibility and convenience.

### Immediate Access

**No Downloads Required**:
- Access instantly from any modern browser
- No app store approvals or updates
- No storage space consumption on devices
- Always running the latest version
- Cross-platform compatibility

**Quick Start**:
- New users can register and start playing within minutes
- Zero friction onboarding
- No waiting for large downloads
- Immediate gameplay access

<Callout type="success">
**Platform Agnostic**: Works on Windows, macOS, Linux, iOS, Android, and any device with a modern web browser.
</Callout>

### Cross-Device Play

A responsive design ensures the interface adapts to different screen sizes and devices.

**Seamless Experience**:
- **Desktop/Laptop**: Full-featured experience with optimal layout
- **Tablet**: Touch-optimized interface with gesture support
- **Mobile**: Streamlined UI for smaller screens
- **Continuous Play**: Start on one device, continue on another
- **Cloud Sync**: Progress automatically saved across all devices

**Device-Specific Optimizations**:
- **Touch Controls**: Natural gestures for mobile and tablet
- **Mouse & Keyboard**: Precise control on desktop
- **Gamepad Support**: Optional controller integration
- **Screen Rotation**: Landscape and portrait modes
- **Smart Scaling**: Adaptive UI elements for all screen sizes

### Low Hardware Requirements

Because heavy computations are offloaded to cloud servers, Mahjong Stars runs smoothly on devices with modest specifications.

**Minimum Requirements**:
- **Browser**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **RAM**: 2GB minimum, 4GB recommended
- **Internet**: 2 Mbps minimum, 5 Mbps recommended
- **CPU**: Any modern dual-core processor
- **Graphics**: Basic GPU or integrated graphics

**Performance Benefits**:
- Older devices can run the platform
- No need for gaming hardware
- Reduced power consumption
- Lower barrier to entry
- Wider market accessibility

<Callout type="info">
**Cloud-Powered**: Server-side processing means your device only handles display and input, making the platform accessible to virtually any modern device.
</Callout>

## Account Creation and Wallet Integration

Creating an account on Mahjong Stars is straightforward with multiple options to suit different user preferences.

### Standard Sign-Up Methods

**Email & Password**:
- Traditional registration with email verification
- Strong password requirements
- Password recovery options
- Email notification preferences

**Social Media Login**:
- **Google**: Quick OAuth integration
- **Facebook**: Connect existing account
- **Apple**: Privacy-focused sign-in
- **Twitter/X**: Social gaming integration
- **Discord**: Community-first approach

**Benefits of Social Login**:
- Faster registration process
- No password to remember
- Easier account recovery
- Social graph integration
- Trust through established platforms

### Wallet Setup (Optional)

Although soft currency accounts are created automatically, players who wish to trade digital assets or interact with the token ecosystem can link a compatible wallet.

<Callout type="warning">
**Entirely Optional**: Wallet integration is NOT required to play Mahjong Stars. It's only needed for blockchain features like NFT trading and token transactions.
</Callout>

**Supported Wallets**:
- **MetaMask**: Most popular Ethereum wallet
- **Trust Wallet**: Mobile-friendly multi-chain wallet
- **Coinbase Wallet**: Beginner-friendly option
- **WalletConnect**: Universal wallet connection
- **Native Web Wallet**: Built-in option for simplicity

**Wallet Setup Process**:
1. Create account with email or social login
2. Navigate to wallet settings (optional)
3. Choose wallet provider
4. Connect existing wallet or create new
5. Verify connection
6. Start using blockchain features

**In-Platform Guides**:
- Step-by-step wallet tutorials
- Video walkthroughs
- Security best practices
- Hardware wallet recommendations
- Recovery phrase importance education

### Security Features

Account security is paramount at Mahjong Stars.

**Security Measures**:
- **Two-Factor Authentication (2FA)**: SMS or authenticator app
- **Email Verification**: Confirm email for account actions
- **Login Alerts**: Notifications for new device logins
- **Session Management**: View and revoke active sessions
- **Biometric Login**: Fingerprint/Face ID on mobile
- **Security Logs**: Track account activity
- **Password Requirements**: Strong password enforcement

**Best Practices Encouraged**:
- Use unique passwords
- Enable 2FA immediately
- Don't share account credentials
- Use hardware wallets for large holdings
- Regular password updates
- Secure recovery phrase storage

## Currency Options

Mahjong Stars employs a dual currency system designed to accommodate all player types.

### Soft Currency (In-Game Credits)

The in-game credit that supports everyday gameplay, accessible to all players without crypto requirements.

<Callout type="success">
**Free-to-Earn**: Soft currency is primarily earned through gameplay, making Mahjong Stars genuinely free-to-play.
</Callout>

**Earning Soft Currency**:
- **Winning Matches**: Primary earning method
- **Daily Quests**: Complete objectives for rewards
- **Weekly Challenges**: Longer-term goals
- **Achievements**: Milestone rewards
- **Tournaments**: Competition prizes
- **Login Bonuses**: Daily rewards for active players
- **AI Earnings**: Your AI earns while you're offline
- **Charm Bonuses**: Increased earning rates

**Using Soft Currency**:
- Purchase characters from primary store
- Buy charms and cosmetics
- Enter tournaments (entry fees)
- Accelerate AI training
- Customize studios
- Purchase theme packs
- Raffle ticket purchases
- Table customization

**Soft Currency Properties**:
- Cannot be directly withdrawn as fiat
- Non-speculative design
- Fixed exchange rate for purchases
- Prevents pay-to-win mechanics
- Inflation-protected

### Hard Currency (Governance Token)

The blockchain-based token for players who want to participate in the platform's governance and marketplace features.

**Token Use Cases**:
- **Governance Voting**: Participate in platform decisions
- **Marketplace Transactions**: Trade assets peer-to-peer
- **Staking**: Earn rewards and unlock benefits
- **Premium Features**: Access exclusive content
- **AI Marketplace**: Buy/sell trained AI models
- **Tournament Entry**: High-stakes competitions
- **Asset Purchases**: Special collectibles

**Acquiring Tokens**:
- Purchase from exchanges
- Earn through high-level tournaments
- Staking rewards
- Governance participation rewards
- Marketplace trading profits
- Special events and airdrops

<Callout type="info">
**Separate Entity**: The governance token is issued and managed by a separate entity, allowing Mahjong Stars to operate the game independently of crypto regulations.
</Callout>

### Deposits and Purchases

**Buying Soft Currency**:
- **Fiat Payment**: Credit/debit cards, PayPal
- **Token Conversion**: Exchange tokens for soft currency
- **Package Deals**: Various denominations available
- **Bonus Offers**: First-time buyer bonuses
- **Secure Payments**: PCI-compliant payment processing

**Payment Options**:
- Credit cards (Visa, Mastercard, Amex)
- Debit cards
- PayPal
- Apple Pay / Google Pay
- Cryptocurrency (via token conversion)
- Regional payment methods (Alipay, WeChat Pay, etc.)

**Security**:
- Encrypted payment processing
- Fraud detection systems
- Buyer protection policies
- Refund procedures
- Secure transaction history

### Redemptions and Withdrawals

**Soft Currency Redemption**:
Soft currency is primarily meant for in-game use and cannot be withdrawn directly as fiat. However, players have indirect redemption options:

**Redemption Path**:
1. Use soft currency to purchase digital assets (characters, charms)
2. Convert assets to NFTs (if blockchain enabled)
3. Sell assets on marketplace for tokens or fiat
4. Use third-party exchanges for final conversion

**This Model**:
- Prevents potential abuse
- Maintains healthy economy
- Provides value extraction path
- Requires player choice and effort
- Complies with regulations

<Callout type="warning">
**Not a Cash-Out System**: Mahjong Stars is a gaming platform, not a financial instrument. Redemption is possible through asset value, but not designed as primary motivation.
</Callout>

### Currency Conversion

**Soft Currency ↔ Token**:
- Convert tokens to soft currency at market rates
- One-way conversion (tokens → soft currency only)
- Prevents token deflation
- Maintains soft currency stability
- Clear exchange rates displayed

**Rate Determination**:
- Market-based pricing
- Transparent calculation
- No hidden fees
- Real-time updates
- Historical rate tracking
